\subsection{Philosophy}

The project follows the principle:
\begin{center}
  \textit{Propositions are types. Proofs are terms.}
\end{center}
Every mathematical statement in Lean is a \texttt{Prop} (a type), and proving it
means constructing a term of that type. The compiler \emph{is} the verifier ---
if Lean accepts the code with no errors and no \texttt{sorry} warnings, the proof
is formally and absolutely correct.

\subsection{Golden Rule}

\begin{tcolorbox}[colback=mathgreen!6, colframe=mathgreen!50]
  Before defining any new mathematical object or concept,
  \textbf{always check whether it already exists in Mathlib}.
  Use \texttt{\#check}, \texttt{\#print}, and the
  \href{https://leanprover-community.github.io/mathlib4_docs/}{online docs}.
\end{tcolorbox}

\subsection{Key Lean Syntax Reference}

\begin{center}
\begin{tabular}{@{}lll@{}}
  \toprule
  \textbf{Symbol} & \textbf{Meaning} & \textbf{Example} \\
  \midrule
  \texttt{:}  & ``has type'' & \texttt{easy : 2 + 2 = 4} \\
  \texttt{:=} & ``is defined as'' & \texttt{:= rfl} \\
  \texttt{=}  & mathematical equality (a \texttt{Prop}) & \texttt{2 + 2 = 4} \\
  \texttt{def} & name a type or function & \texttt{def FermatLittleTheorem := \ldots} \\
  \texttt{structure} & define a multi-field object & domains, norms, etc. \\
  \texttt{class} & shared properties across types & normed spaces, etc. \\
  \bottomrule
\end{tabular}
\end{center}

\section{Chapter 1: Transport Equation (Evans §2.1)}
% ══════════════════════════════════════════════════════════════════════════════

\subsection{§2.1.1 Homogeneous Case}

The homogeneous transport equation is:
\begin{equation}
  u_t + b \cdot Du = 0 \quad \text{in } \mathbb{R}^n \times (0, \infty),
  \label{eq:transport}
\end{equation}
with initial condition $u = g$ on $\mathbb{R}^n \times \{t = 0\}$.
Here $b \in \mathbb{R}^n$ is a constant vector and $g : \mathbb{R}^n \to \mathbb{R}$
is given initial data. By the method of characteristics, the unique $C^1$ solution is:
\[
  u(x, t) = g(x - tb).
\]

\subsubsection{Definitions}

The PDE predicate uses \texttt{spatialGradient} and \texttt{timeDerivative}
from \texttt{Common/\allowbreak Calculus.\allowbreak lean}, matching Evans' notation directly:

\begin{lstlisting}[caption={PDE predicate}]
def IsTransportSolution (b : Rn) (u : Rn x R -> R) : Prop :=
  ForAll p : Rn x R,
    timeDerivative u p + <spatialGradient u p, b>_R = 0
\end{lstlisting}

The characteristic flow $(x,t) \mapsto x - tb$ is a continuous linear map.
Two simp lemmas are registered so that Lean can compute with it automatically:

\begin{lstlisting}[caption={Characteristic flow}]
noncomputable def charFlow (b : Rn) : Rn x R ->L[R] Rn :=
  ContinuousLinearMap.fst R Rn R -
  (ContinuousLinearMap.snd R Rn R).smulRight b

@[simp]
lemma charFlow_apply (b x : Rn) (t : R) :
    charFlow b (x, t) = x - t * b := by simp [charFlow]

lemma charFlow_direction_zero (b : Rn) :
    charFlow b (b, 1) = 0 := by simp
\end{lstlisting}

The solution formula is the composition of $g$ with the characteristic flow:

\begin{lstlisting}[caption={Evans formula}]
noncomputable def evansFormula (b : Rn) (g : Rn -> R) : Rn x R -> R :=
  g o charFlow b

@[simp]
lemma evansFormula_apply (b : Rn) (g : Rn -> R) (x : Rn) (t : R) :
    evansFormula b g (x, t) = g (x - t * b) := by simp [evansFormula]
\end{lstlisting}

The \texttt{@[simp]} attribute registers these as rewrite rules, so \texttt{simp}
can unfold \texttt{charFlow} and \texttt{evansFormula} automatically in proofs.

\subsubsection{Results}

\begin{lstlisting}[caption={Initial condition and regularity (proved)}]
theorem evansFormula_initial (b : Rn) (g : Rn -> R) (x : Rn) :
    evansFormula b g (x, 0) = g x

theorem evansFormula_differentiable (b : Rn) (g : Rn -> R)
    (hg : Differentiable R g) :
    Differentiable R (evansFormula b g)
\end{lstlisting}

\begin{proof}[Initial condition]
By definition $u(x,t) = g(x - tb)$, so $u(x,0) = g(x - 0\cdot b) = g(x)$.
\end{proof}

\begin{proof}[Regularity]
The solution is the composition $u = g\circ\Phi$, where $\Phi(x,t) = x - tb$ is an affine
(in particular smooth) map. By the chain rule a composition of differentiable maps is
differentiable, so $u$ is differentiable whenever $g$ is.
\end{proof}

\begin{lstlisting}[caption={Spatial gradient pulls back along charFlow (proved)}]
lemma spatialGradient_evansFormula (b : Rn) (g : Rn -> R)
    (hg : Differentiable R g) (p : Rn x R) :
    spatialGradient (evansFormula b g) p = gradient g (charFlow b p) := by
  simp only [spatialGradient, evansFormula, Function.comp, charFlow_apply]
  ... -- chain rule through the affine charFlow; fully proved in Transport.lean
\end{lstlisting}

\begin{lstlisting}[caption={Time derivative equals minus inner product with b (proved)}]
lemma timeDerivative_evansFormula (b : Rn) (g : Rn -> R)
    (hg : Differentiable R g) (p : Rn x R) :
    timeDerivative (evansFormula b g) p = -<gradient g (charFlow b p), b>_R := by
  simp only [timeDerivative, evansFormula, Function.comp, charFlow_apply]
  ... -- chain rule; fully proved in Transport.lean
\end{lstlisting}

Both follow from the chain rule: $x \mapsto x - tb$ has spatial derivative the identity,
so the spatial gradient of $g(x-tb)$ is just $\nabla g(x-tb)$; and
$\partial_t[g(x-tb)] = \nabla g(x-tb) \cdot (-b) = -\langle \nabla g(x-tb), b \rangle$.
Substituting these two identities into $u_t + b\cdot Du$ gives
$-\langle\nabla g, b\rangle + \langle b, \nabla g\rangle = 0$, so the formula solves the
transport equation:

\begin{lstlisting}[caption={Existence (proved)}]
theorem evansFormula_solves_transport (b : Rn) (g : Rn -> R)
    (hg : Differentiable R g) :
    IsTransportSolution b (evansFormula b g) := by
  intro p
  rw [timeDerivative_evansFormula b g hg p,
      spatialGradient_evansFormula b g hg p]
  simp [real_inner_comm]
\end{lstlisting}

\begin{lstlisting}[caption={Uniqueness via characteristics (proved)}]
theorem evansFormula_unique (b : Rn) (g : Rn -> R)
    (u : Rn x R -> R)
    (hu_pde  : IsTransportSolution b u)
    (hu_init : ForAll x : Rn, u (x, 0) = g x)
    (hu_diff : Differentiable R u) :
    u = evansFormula b g := by
  -- proved: see Transport.lean for full proof
  -- key: z(s) = u(x+sb, t+s) has z' = 0 by chain rule + PDE,
  -- so z is constant; z(-t) = g(x-tb) gives u(x,t) = g(x-tb)
\end{lstlisting}

\begin{proof}[Uniqueness]
Let $u$ be any differentiable solution with $u(\cdot,0)=g$, and fix $(x,t)$. Consider the
function along the characteristic through $(x,t)$,
\[
  z(s) = u(x+sb,\ t+s).
\]
By the chain rule its derivative is the directional derivative of $u$ at $(x+sb,t+s)$ in the
direction $(b,1)$, namely $\langle Du,\,b\rangle + u_t$, which vanishes by the transport
equation. A function on $\mathbb{R}$ with identically zero derivative is constant (mean value
theorem), so $z$ is constant. Evaluating at $s=-t$ gives $z(-t) = u(x-tb,0) = g(x-tb)$, and
hence $u(x,t) = z(0) = g(x-tb)$. As $(x,t)$ was arbitrary, $u$ equals the explicit solution.
\end{proof}

\subsection{§2.1.2 Inhomogeneous Case}

The inhomogeneous transport equation introduces a source term $f$:
\begin{equation}
  u_t + b \cdot Du = f \quad \text{in } \mathbb{R}^n \times (0, \infty),
  \label{eq:inhomog_transport}
\end{equation}
with initial condition $u = g$ on $\mathbb{R}^n \times \{t = 0\}$.
By Duhamel's principle, the solution is:
\begin{equation}
  u(x, t) = g(x - tb) + \int_0^t f(x - (t-s)b,\ s)\, ds.
  \label{eq:duhamel}
\end{equation}
The first term solves the homogeneous equation; the integral corrects for the source.

\subsubsection{Definitions}

\begin{lstlisting}[caption={Inhomogeneous PDE predicate}]
def IsInhomTransportSolution (b : Rn) (f : Rn x R -> R)
    (u : Rn x R -> R) : Prop :=
  ForAll p : Rn x R,
    timeDerivative u p + <spatialGradient u p, b>_R = f p
\end{lstlisting}

\begin{lstlisting}[caption={Duhamel's formula}]
noncomputable def duhamelFormula (b : Rn) (g : Rn -> R)
    (f : Rn x R -> R) : Rn x R -> R :=
  fun p => g (p.1 - p.2 * b) +
           integral s in 0..p.2, f (p.1 - (p.2 - s) * b, s)
\end{lstlisting}

The integral uses Mathlib's interval integral \texttt{intervalIntegral}
(\texttt{∫ s in a..b, ...}).

\subsubsection{Results}

\begin{lstlisting}[caption={Initial condition: integral vanishes at t=0}]
theorem duhamelFormula_initial (b : Rn) (g : Rn -> R)
    (f : Rn x R -> R) (x : Rn) :
    duhamelFormula b g f (x, 0) = g x
\end{lstlisting}

\begin{proof}
At $t = 0$, the integral term $\int_0^0 f(x-(0-s)b, s)\,ds = 0$ since the
integration domain is empty. The remaining term is $g(x - 0 \cdot b) = g(x)$.
\end{proof}

\begin{lstlisting}[caption={Existence (proved)}]
theorem duhamelFormula_solves (b : Rn) (g : Rn -> R)
    (f : Rn x R -> R) (hg : Differentiable R g)
    (hf : ContDiff R 1 f) :
    IsInhomTransportSolution b f (duhamelFormula b g f) := by
  -- Step 1 (proved): partial_t[f(x-(t-s)b,s)] = -<grad f, b> via chain rule
  -- Step 2 (proved): Leibniz rule via Calculus.leibniz_integral
  -- Step 3 (proved): time derivative of g(x-tb) via chain rule
  -- Step 4 (proved): hg_deriv + hleibniz via HasDerivAt.add
  -- Step 5 (proved): spatial gradient via
  --   hasFDerivAt_integral_of_dominated_loc_of_lip; spatial LipschitzOnWith
  --   via the mean value inequality with a uniform fderiv bound over a compact tube
  -- Step 6 (proved): <Int grad f, b> = Int <grad f, b>
  --   via ContinuousLinearMap.intervalIntegral_comp_comm
  ... -- fully proved, no sorry
\end{lstlisting}

\begin{proof}
Write $u = v + w$, where $v(x,t) = g(x-tb)$ solves the homogeneous equation
$v_t + b\cdot Dv = 0$ with $v(\cdot,0)=g$ (the previous case), and
\[
  w(x,t) = \int_0^t f\bigl(x-(t-s)b,\ s\bigr)\,ds .
\]
Differentiate $w$ in $t$ by the Leibniz rule for an integral with a \emph{variable} upper
limit. The moving endpoint contributes the boundary term $f\bigl(x-(t-t)b,\ t\bigr)=f(x,t)$,
and differentiating the integrand contributes, by the chain rule,
\[
  \int_0^t \partial_t\!\bigl[f(x-(t-s)b,s)\bigr]\,ds
    = -\int_0^t b\cdot\nabla f(x-(t-s)b,s)\,ds .
\]
The spatial term, again by the chain rule and differentiation under the integral sign, is
\[
  b\cdot Dw(x,t) = \int_0^t b\cdot\nabla f(x-(t-s)b,s)\,ds .
\]
Adding, the two integrals cancel and $w_t + b\cdot Dw = f(x,t)$. Since $w(\cdot,0)=0$, the
sum $u = v+w$ satisfies $u_t + b\cdot Du = f$ with $u(\cdot,0)=g$.

The one ingredient not directly available off the shelf is the Leibniz rule with a variable
upper limit,
\[
  \frac{d}{dt'}\int_0^{t'} H(t',s)\,ds\Big|_{t'=t}
    = H(t,t) + \int_0^t \partial_1 H(t,s)\,ds,
\]
which we prove by splitting $F(t')=\int_0^{t'}H(t',s)\,ds$ as $P+B+C$: the fixed-integrand
piece $P(t')=\int_0^{t'}H(t,s)\,ds$ (handled by the fundamental theorem of calculus, giving
$H(t,t)$); the fixed-interval piece $B(t')=\int_0^t\bigl(H(t',s)-H(t,s)\bigr)\,ds$
(differentiation under the integral over $[0,t]$, giving $\int_0^t\partial_1 H(t,s)\,ds$);
and a cross-term $C(t')=\int_t^{t'}\bigl(H(t',s)-H(t,s)\bigr)\,ds = O\bigl((t'-t)^2\bigr)$,
whose derivative at $t$ is $0$. The uniform bounds these steps require follow from continuity
of $\partial_1 H$ on a compact box.
\end{proof}

\section{Chapter 2: Laplace and Poisson Equations (Evans §2.2)}
% ══════════════════════════════════════════════════════════════════════════════

\subsection{The PDEs}

The two equations covered in this section are:
\begin{align}
  -\Delta u &= 0 \quad \text{in } U \subseteq \mathbb{R}^n \quad \text{(Laplace)} \\
  -\Delta u &= f \quad \text{in } U \subseteq \mathbb{R}^n \quad \text{(Poisson)}
\end{align}
where $U$ is an open set. A solution to Laplace's equation is called \textbf{harmonic}.
The Laplacian $\Delta$ is Mathlib's \texttt{Laplacian.laplacian}, opened as \texttt{Δ}.

\begin{lstlisting}[caption={PDE predicates}]
def IsHarmonic (U : Set Rn) (u : Rn -> R) : Prop :=
  ForAll x in U, Delta u x = 0

def IsPoissonSolution (U : Set Rn) (f : Rn -> R) (u : Rn -> R) : Prop :=
  ForAll x in U, -Delta u x = f x
\end{lstlisting}

\begin{lstlisting}[caption={Laplace is Poisson with f=0 (proved)}]
lemma isHarmonic_iff_isPoissonSolution_zero (U : Set Rn) (u : Rn -> R) :
    IsHarmonic U u <-> IsPoissonSolution U 0 u
\end{lstlisting}

\begin{proof}
$u$ is harmonic if and only if $\Delta u = 0$ on $U$, which holds if and only if
$-\Delta u = 0 = f$ on $U$, i.e., $u$ solves Poisson's equation with $f = 0$.
\end{proof}

\subsection{§2.2.1 Fundamental Solution}

\begin{definition}
The \textbf{fundamental solution} $\Phi : \mathbb{R}^n \setminus \{0\} \to \mathbb{R}$
satisfies $-\Delta \Phi = \delta_0$ in the distributional sense, where $\omega_n$
denotes the volume of the unit ball in $\mathbb{R}^n$:
\[
  \Phi(x) = \begin{cases}
    \dfrac{1}{n(n-2)\omega_n} |x|^{2-n} & n \geq 3 \\[6pt]
    -\dfrac{1}{2\pi} \log|x| & n = 2
  \end{cases}
\]
\end{definition}

\begin{lstlisting}[caption={Fundamental solution smooth off zero (proved)}]
lemma fundamentalSolution_contDiff_off_zero :
    ContDiffOn R Top fundamentalSolution ({0} : Set Rn)^c
\end{lstlisting}

\begin{proof}
We check each case separately. For $n = 0$, $\Phi = 0$ is trivially smooth.
For $n = 1$, $\Phi(x) = \frac{1}{2}\|x\|$, which is smooth since $\|x\|$ is
$C^\infty$ on $\mathbb{R} \setminus \{0\}$.
For $n = 2$, $\Phi(x) = c\log\|x\|$. Since $\|x\| > 0$ for $x \neq 0$, and
$\log$ is $C^\infty$ on $(0,\infty)$, the composition is $C^\infty$.
For $n \geq 3$, $\Phi(x) = c\|x\|^{2-n}$. Since $\|x\| \neq 0$ for $x \neq 0$,
and $t \mapsto t^{2-n}$ is $C^\infty$ on $(0,\infty)$, the composition is $C^\infty$.
\end{proof}

The \textbf{Newtonian potential} is the convolution of $\Phi$ with a source $f$:

\begin{lstlisting}[caption={Newtonian potential}]
noncomputable def newtonianPotential (f : Rn -> R) : Rn -> R :=
  fun x => integral y, fundamentalSolution (x - y) * f y
\end{lstlisting}

\subsection{§2.2.1 Harmonicity of the Fundamental Solution}

This subsection contains the key analytic computation: proving $\Delta\Phi = 0$
away from the origin. The proof requires three helper lemmas.

\subsubsection{Helper definitions}

To avoid type ambiguity with Mathlib's conjugate-linear \texttt{innerSL}, we

lemma realInnerL_apply (x y : Rn) : realInnerL x y = <x, y>_R
\end{lstlisting}

\subsubsection{Helper lemmas (all proved)}

\begin{lstlisting}[caption={Scalar linearity of Laplacian (proved)}]
lemma laplacian_const_mul (c : R) (f : Rn -> R) (hf : ContDiffAt R 2 f x) :
    Delta (fun y => c * f y) x = c * Delta f x
\end{lstlisting}

\begin{proof}
Since $\Delta$ is a linear operator, $\Delta(c \cdot f) = c \cdot \Delta f$
for any constant $c \in \mathbb{R}$ and $C^2$ function $f$.
\end{proof}

\begin{lstlisting}[caption={First derivative of radial power (proved)}]
lemma hasFDerivAt_norm_rpow_of_ne (x : Rn) (hx : x != 0) (p : R) :
    HasFDerivAt (fun x => norm x ^ p)
      ((p * norm x ^ (p - 2)) * realInnerL x) x
\end{lstlisting}

\begin{proof}
Write $\|x\|^p = (\|x\|^2)^{p/2}$. By the chain rule:
\[
  D(\|x\|^p)(x)(v) = \frac{p}{2}(\|x\|^2)^{p/2 - 1} \cdot D(\|x\|^2)(x)(v)
  = \frac{p}{2}\|x\|^{p-2} \cdot 2\langle x, v\rangle = p\|x\|^{p-2}\langle x, v\rangle.
\]
Hence the Fréchet derivative at $x$ is the linear map $v \mapsto p\|x\|^{p-2}\langle x, v\rangle$.
\end{proof}

\begin{lstlisting}[caption={Laplacian of radial power (proved)}]
lemma laplacian_norm_rpow_eq (p : R) (x : Rn) (hx : x != 0) :
    Delta (fun x => norm x ^ p) x = p * (n + p - 2) * norm x ^ (p - 2)
\end{lstlisting}

\begin{proof}
The gradient is $\nabla(\|x\|^p) = p\|x\|^{p-2}x$ (chain rule, using
$\nabla\|x\| = x/\|x\|$).
Differentiating again using the product rule:
\[
  D^2(\|x\|^p)(x)(v,w) = p(p-2)\|x\|^{p-4}\langle x,v\rangle\langle x,w\rangle
  + p\|x\|^{p-2}\langle v,w\rangle.
\]
Taking the trace over an orthonormal basis $\{e_i\}$:
\begin{align*}
  \Delta(\|x\|^p)(x)
  &= \sum_i D^2(\|x\|^p)(x)(e_i, e_i) \\
  &= p(p-2)\|x\|^{p-4}\underbrace{\sum_i\langle x,e_i\rangle^2}_{=\|x\|^2}
   + p\|x\|^{p-2}\underbrace{\sum_i\langle e_i,e_i\rangle}_{=n} \\
  &= p(p-2)\|x\|^{p-2} + pn\|x\|^{p-2} = p(n+p-2)\|x\|^{p-2}.
\end{align*}
\end{proof}

\begin{lstlisting}[caption={Laplacian of log norm (proved)}]
lemma laplacian_log_norm_eq (x : Rn) (hx : x != 0) :
    Delta (fun x => log (norm x)) x = (n - 2) * norm x ^ (-2)
\end{lstlisting}

\begin{proof}
The computation parallels that of $\Delta(\|x\|^p)$ above.

\textbf{Step 1: First derivative.}
Write $\log\|y\| = \log(\|y\|^1)$ and apply the chain rule.
Since $D(\|y\|)(y)(v) = \|y\|^{-1}\langle y, v\rangle$, and
$\frac{d}{dt}\log t\big|_{t=\|y\|} = \|y\|^{-1}$, the chain rule gives:
\[
  D(\log\|y\|)(y)(v) = \|y\|^{-1} \cdot \|y\|^{-1}\langle y, v\rangle
  = \|y\|^{-2}\langle y, v\rangle.
\]
Hence $\nabla(\log\|y\|) = \|y\|^{-2}y$, and the first derivative is
the linear map $v \mapsto \|y\|^{-2}\langle y, v\rangle$.

\textbf{Step 2: Second derivative.}
Differentiating $y \mapsto \|y\|^{-2}\langle y, v\rangle$ using the product rule:
\[
  D^2(\log\|x\|)(x)(v,w)
  = -2\|x\|^{-4}\langle x,v\rangle\langle x,w\rangle + \|x\|^{-2}\langle v,w\rangle.
\]

\textbf{Step 3: Trace over orthonormal basis.}
Taking the trace over an orthonormal basis $\{e_i\}$:
\begin{align*}
  \Delta(\log\|x\|)
  &= \sum_i D^2(\log\|x\|)(x)(e_i,e_i) \\
  &= -2\|x\|^{-4}\underbrace{\sum_i\langle x,e_i\rangle^2}_{=\|x\|^2}
   + \|x\|^{-2}\underbrace{\sum_i\langle e_i,e_i\rangle}_{=n} \\
  &= -2\|x\|^{-2} + n\|x\|^{-2} = (n-2)\|x\|^{-2}.
\end{align*}
In particular, when $n = 2$, $\Delta(\log\|x\|) = 0$, confirming that
$\log\|x\|$ is harmonic in $\mathbb{R}^2 \setminus \{0\}$.
\end{proof}

\subsubsection{Main result}

\begin{lstlisting}[caption={Fundamental solution is harmonic off zero (proved modulo log lemma)}]
lemma fundamentalSolution_harmonic_off_zero (x : Rn) (hx : x != 0) :
    Delta fundamentalSolution x = 0
\end{lstlisting}

\begin{proof}
We consider each dimension separately.

\textbf{Case $n = 0$:} $\Phi \equiv 0$, so $\Delta\Phi = 0$ trivially.

\textbf{Case $n = 1$:} $\Phi(x) = \frac{1}{2}\|x\|^1$.
By linearity and the formula $\Delta(\|x\|^p) = p(n+p-2)\|x\|^{p-2}$ with $p = 1$:
\[ \Delta\Phi = \frac{1}{2} \cdot 1 \cdot (1 + 1 - 2) \cdot \|x\|^{-1} = 0. \]

\textbf{Case $n = 2$:} $\Phi(x) = -\frac{1}{2\pi}\log\|x\|$.
By linearity and $\Delta(\log\|x\|) = (n-2)\|x\|^{-2}$:
\[ \Delta\Phi = -\frac{1}{2\pi} \cdot (2-2) \cdot \|x\|^{-2} = 0. \]

\textbf{Case $n \geq 3$:} $\Phi(x) = \frac{1}{n(n-2)\omega_n}\|x\|^{2-n}$.
By linearity and the same formula with $p = 2-n$:
\[ \Delta\Phi = c \cdot (2-n) \cdot (n + (2-n) - 2) \cdot \|x\|^{-n}
             = c \cdot (2-n) \cdot 0 = 0. \]

In all cases $\Delta\Phi(x) = 0$ for $x \neq 0$.
\end{proof}

\subsection{§2.2.3 Maximum Principles and Regularity}

\textbf{\texttt{harmonic\_weakMax} (Evans §2.2.3, Theorem 3) is now proved} for a $C^2$
harmonic function on a bounded open set: it attains its supremum on the boundary,
$u(x) \le \sup_{\partial U} u$ for all $x \in U$. The proof is the classical subharmonic
perturbation. For $\varepsilon > 0$ set $v_\varepsilon = u + \varepsilon\|\cdot\|^2$; then
$\Delta v_\varepsilon = \Delta u + 2n\varepsilon = 2n\varepsilon > 0$, since
$\Delta\|\cdot\|^2 = 2n$. A function with strictly positive Laplacian can have no interior
maximum: at an interior local maximum of a $C^2$ function the Laplacian (the trace of the
Hessian, which is negative semidefinite there) is $\le 0$, so an interior maximum would force
$\Delta v_\varepsilon \le 0$, a contradiction. Hence $v_\varepsilon$
attains its max on $\overline U$ at a boundary point (the max exists by compactness of
$\overline U$ and continuity), giving $u(x) \le \sup_{\partial U} u + \varepsilon R^2$ with
$R = \sup_{\overline U}\|\cdot\|$; letting $\varepsilon \to 0$ finishes.

The remaining two results are not yet formalized. The strong maximum principle and the
interior-regularity theorem both go through the mean value property (Evans §2.2.2). The
Gauss--Green step that the mean value property needs is now available (\texttt{divergence\_theorem},
Chapter~6); what remains is the concrete instantiation of a ball/annulus as an
\texttt{IsBoundedC1Domain} with its outward normal, plus (for the exact flux constant) the sphere
surface-measure formula --- now derivable in principle from the Chapter~6 \texttt{area\_formula}.

\subsection{The PDEs}

The initial value problem for the heat (diffusion) equation:
\begin{align}
  u_t - \Delta u &= 0 \quad \text{in } \mathbb{R}^n \times (0,\infty) \quad \text{(homogeneous)} \\
  u_t - \Delta u &= f \quad \text{in } \mathbb{R}^n \times (0,\infty) \quad \text{(inhomogeneous)} \\
  u &= g \quad \text{on } \mathbb{R}^n \times \{t = 0\}.
\end{align}
Here $u_t$ is \texttt{timeDerivative} and $\Delta u$ is the spatial Laplacian
\texttt{spatialLaplacian}, both from \texttt{Common/\allowbreak Calculus.\allowbreak lean}.

Unlike the Laplace chapter, the heat chapter avoids the three Mathlib blockers
(Stokes on spheres, \texttt{hausdorffMeasure\_sphere}, $n$-dim polar coordinates):
the fundamental solution is an \emph{explicit} Gaussian, and its normalization uses
Mathlib's multivariate Gaussian integral directly.

\subsection{§2.3.1 The Heat Kernel}

The fundamental solution (heat kernel) is
\begin{equation}
  \Phi(x, t) =
  \begin{cases}
    (4\pi t)^{-n/2} \exp\!\left(-\dfrac{|x|^2}{4t}\right) & t > 0, \\[1ex]
    0 & t \le 0.
  \end{cases}
\end{equation}

\begin{lstlisting}[caption={Heat kernel definition}]
noncomputable def heatKernel (p : Rn x R) : R :=
  if 0 < p.2 then
    (4 * pi * p.2) ^ (-n / 2) * exp (-||p.1||^2 / (4 * p.2))
  else 0

noncomputable def heatSolution (g : Rn -> R) : Rn x R -> R :=
  fun p => Int y, heatKernel (p.1 - y, p.2) * g y
\end{lstlisting}

The key computations are the time derivative and spatial Laplacian of the kernel,
which turn out to be \emph{equal}, so the kernel solves the heat equation:
\begin{align}
  \Phi_t(x,t) &= \Phi(x,t) \cdot \left(\frac{|x|^2}{4t^2} - \frac{n}{2t}\right), \\
  \Delta \Phi(x,t) &= \Phi(x,t) \cdot \left(\frac{|x|^2}{4t^2} - \frac{n}{2t}\right).
\end{align}

\textbf{Normalization} ($\int_{\mathbb{R}^n} \Phi(\cdot, t) = 1$): the kernel is
$(4\pi t)^{-n/2}$ times the Gaussian $\exp(-|x|^2/4t)$, whose integral over $\mathbb{R}^n$ is
$(4\pi t)^{n/2}$ (the standard Gaussian integral). The two powers cancel.

\textbf{Time derivative}: for $t > 0$ the kernel agrees with the smooth branch
$F(s) = (4\pi s)^{-n/2} \exp(-|x|^2/4s)$; writing $\Phi = u\cdot v$ with
$u(s) = (4\pi s)^{-n/2}$ and $v(s) = \exp(-|x|^2/4s)$, the product and chain rules give the
stated formula.

\textbf{Spatial Laplacian}: expanding over the standard orthonormal basis, the second
directional derivative of the Gaussian along each $e_i$ is computed by the product rule, and
summing gives $\Delta$ using $\sum_i \langle x, e_i\rangle^2 = |x|^2$; the constant
$(4\pi t)^{-n/2}$ factors out by linearity. The result coincides with the time derivative.

\begin{proof}
The cancellation is the heart of the matter. For each fixed $y$, the translated kernel
$(x,t)\mapsto\Phi(x-y,t)$ again solves the heat equation: the Laplacian is
translation-invariant, and $\Phi$ itself solves the equation, so the translate's time
derivative equals its spatial Laplacian. Consequently, \emph{provided both derivatives may be
taken under the integral sign},
\[
  u_t(x,t) - \Delta u(x,t)
  = \int_{\mathbb{R}^n}\bigl[\partial_t\Phi(x-y,t) - \Delta_x\Phi(x-y,t)\bigr]g(y)\,dy = 0,
\]
the integrand vanishing pointwise.

It remains to justify differentiating under the integral. We assume, as in Evans, that $g$
is \emph{bounded} continuous ($|g|\le C_g$) --- the growth condition that makes the
convolution and its derivatives converge. Both differentiations are justified by domination
on a compact time window $(t/2,2t)\ni t$. The relevant derivatives of the kernel ---
$\partial_t\Phi$, and the first and second spatial derivatives $D_x\Phi$, $D^2_x\Phi$, all
available in closed form --- are Gaussians multiplied by polynomials in $x-y$, hence dominated
by integrable functions whose integrability reduces (after the substitution $y\mapsto x-y$) to
the zeroth and second Gaussian moments. The spatial Laplacian is obtained as the sum over the
basis of the second directional derivatives $\int g\,\partial_{e_i}^2\Phi$. With domination in
hand, both $u_t = \int g\,\Phi_t$ and $\Delta u = \int g\,\Delta\Phi$ are valid, and their
integrands cancel.
\end{proof}

\subsection{§2.3.3–2.3.4 Maximum Principle and Uniqueness}

On a bounded space–time cylinder, a $C^2$ solution of the heat equation is controlled by its
values on the \textbf{parabolic boundary}
\[
  \Gamma_T = \bigl(\overline U \times \{0\}\bigr) \cup \bigl(\partial U \times [0,T]\bigr),
\]
the bottom of the cylinder together with its lateral side (but \emph{not} the top
$\overline U \times \{T\}$).

\begin{theorem}[Weak maximum principle]
Let $U\subseteq\mathbb{R}^n$ be bounded open and $T>0$. If $u\in C^2$ solves $u_t=\Delta u$,
then for every $x\in U$ and $0<t\le T$,
\[
  u(x,t)\ \le\ \sup_{\Gamma_T} u .
\]
\end{theorem}

\begin{proof}
Fix $\varepsilon>0$ and set $v_\varepsilon(x,t)=u(x,t)-\varepsilon t$. Then
$\partial_t v_\varepsilon-\Delta v_\varepsilon=(u_t-\Delta u)-\varepsilon=-\varepsilon<0$.
The closed cylinder $\overline U\times[0,T]$ is compact and $v_\varepsilon$ is continuous, so
$v_\varepsilon$ attains a maximum there, at some point $(x_0,t_0)$. Suppose this maximum point
were \emph{not} on $\Gamma_T$, i.e.\ $x_0\in U$ and $t_0\in(0,T]$. Then:
\begin{itemize}
  \item the spatial slice $x\mapsto v_\varepsilon(x,t_0)$ has an interior maximum at $x_0$, so
    $\Delta v_\varepsilon(x_0,t_0)\le 0$ (the Laplacian is the trace of a negative semidefinite
    Hessian at an interior maximum);
  \item the time slice $s\mapsto v_\varepsilon(x_0,s)$ has a maximum on $[0,t_0]$ at the right
    end $t_0$, so its derivative there is $\ge 0$: the left difference quotients
    $(v_\varepsilon(x_0,s)-v_\varepsilon(x_0,t_0))/(s-t_0)$ are $\ge 0$ for $s<t_0$, and pass to
    the limit. Hence $\partial_t v_\varepsilon(x_0,t_0)\ge 0$.
\end{itemize}
Therefore $\partial_t v_\varepsilon-\Delta v_\varepsilon\ge 0$ at $(x_0,t_0)$, contradicting
$=-\varepsilon<0$. So the maximum of $v_\varepsilon$ lies on $\Gamma_T$, and for any $(x,t)$ in
the cylinder, $v_\varepsilon(x,t)\le\max_{\Gamma_T}v_\varepsilon\le\sup_{\Gamma_T}u$ (as
$v_\varepsilon\le u$). Thus $u(x,t)\le\sup_{\Gamma_T}u+\varepsilon t\le\sup_{\Gamma_T}u+\varepsilon T$;
letting $\varepsilon\to0$ gives the claim.
\end{proof}

\begin{theorem}[Uniqueness on a cylinder]
Two $C^2$ solutions of the heat equation that agree on the parabolic boundary $\Gamma_T$ agree
throughout the cylinder.
\end{theorem}

\begin{proof}
The difference $w=u_1-u_2$ is again a $C^2$ heat solution (the heat operator is linear), and
$w=0$ on $\Gamma_T$. Applying the maximum principle to $w$ gives $w\le\sup_{\Gamma_T}w=0$, and to
$-w=u_2-u_1$ gives $-w\le 0$; hence $w\equiv 0$ on the cylinder.
\end{proof}

\section{Chapter 4: Wave Equation (Evans §2.4)}
% ══════════════════════════════════════════════════════════════════════════════

\subsection{The PDE}

The one-dimensional wave equation initial value problem:
\begin{align}
  u_{tt} - u_{xx} &= 0 \quad \text{in } \mathbb{R} \times (0,\infty) \\
  u = g, \quad u_t &= h \quad \text{on } \mathbb{R} \times \{t = 0\}.
\end{align}
We work on spacetime $\mathbb{R} \times \mathbb{R}$ with ordinary second derivatives
$u_{tt}$, $u_{xx}$ (\texttt{timeDeriv2}, \texttt{spaceDeriv2}), since in one space dimension
there is no need for the gradient/Laplacian machinery.

\subsection{§2.4.1 Traveling Waves and d'Alembert's Formula}

The structural fact: the general solution is a superposition of a right-moving wave
$\varphi(x-t)$ and a left-moving wave $\psi(x+t)$. d'Alembert's formula
\begin{equation}
  u(x,t) = \tfrac{1}{2}\bigl(g(x+t) + g(x-t)\bigr)
    + \tfrac{1}{2}\int_{x-t}^{x+t} h(s)\,ds
\end{equation}
is exactly such a superposition.

\begin{lstlisting}[caption={Wave equation definitions}]
noncomputable def timeDeriv2 (u : R x R -> R) (p : R x R) : R :=
  deriv (fun s => deriv (fun r => u (p.1, r)) s) p.2

noncomputable def spaceDeriv2 (u : R x R -> R) (p : R x R) : R :=
  deriv (fun s => deriv (fun r => u (r, p.2)) s) p.1

def IsWaveSolution (u : R x R -> R) : Prop :=
  forall p, timeDeriv2 u p = spaceDeriv2 u p

noncomputable def dalembert (g h : R -> R) : R x R -> R :=
  fun p => (g (p.1 + p.2) + g (p.1 - p.2)) / 2
            + (Int s in (p.1 - p.2)..(p.1 + p.2), h s) / 2
\end{lstlisting}

\textbf{Traveling waves} (all proved): if $\varphi \in C^2$, then $\varphi(x+t)$ and
$\varphi(x-t)$ each solve $u_{tt} = u_{xx}$. By the chain rule applied to the affine shifts
$x\mapsto x\pm t$, both $u_{tt}$ and $u_{xx}$ equal $\varphi''(x\pm t)$, so they coincide. By
superposition, a sum $\Phi(x+t)+\Psi(x-t)$ of a forward and a backward wave is also a solution,
both of its second derivatives equalling $\Phi''(x+t) + \Psi''(x-t)$.

\textbf{Initial position} (proved): $u(x,0) = g(x)$, since the integral term vanishes
when the limits coincide.

\textbf{Initial velocity} (proved): $u_t(x,0) = h(x)$. The $g$-terms' time derivatives
cancel at $t=0$, while the integral term contributes $\tfrac{1}{2}(h(x)+h(x)) = h(x)$ via
the fundamental theorem of calculus (writing the interval integral as $H(x+t)-H(x-t)$ for
an antiderivative $H$, then differentiating).

\textbf{d'Alembert solves the wave equation} (proved): with $H(a) = \int_0^a h$, the
splitting $\int_{x-t}^{x+t} h = H(x+t) - H(x-t)$ rewrites d'Alembert as the traveling-wave
superposition $\Phi(x+t) + \Psi(x-t)$ with $\Phi = (g+H)/2$, $\Psi = (g-H)/2$. Both are
$C^2$ (since $H' = h \in C^1$), so the traveling-wave result above applies directly.

This makes \textbf{Wave §2.4.1 the first fully closed chapter} --- every result is proved
with no \texttt{sorry}, not even technical ones, since the one-dimensional setting needs
only ordinary derivatives and the fundamental theorem of calculus, all available in Mathlib.

\subsection{§2.4.3 Energy Methods}

For a wave solution $u$, the energy density $e = \tfrac{1}{2}(u_t^2 + u_x^2)$ and flux
$p = u_t \cdot u_x$ satisfy the local conservation law $\partial_t e = \partial_x p$.
Integrating over space gives conservation of total energy $\frac{d}{dt}\int e = 0$, the
basis for uniqueness and finite propagation speed.

\begin{lstlisting}[caption={Energy density and flux}]
noncomputable def energyDensity (u : R x R -> R) (p : R x R) : R :=
  ((timeDeriv u p)^2 + (spaceDeriv u p)^2) / 2

noncomputable def energyFlux (u : R x R -> R) (p : R x R) : R :=
  timeDeriv u p * spaceDeriv u p
\end{lstlisting}

\textbf{Local energy conservation} (proved): for $\Phi, \Psi \in C^2$, the wave solution
$u = \Phi(x+t) + \Psi(x-t)$ satisfies $\partial_t e = \partial_x p$. The proof computes
both energy density and flux as traveling-wave superpositions in the profiles
$(\Phi')^2$ and $(\Psi')^2$:
\begin{align}
  e &= (\Phi'(x+t))^2 + (\Psi'(x-t))^2, \\
  p &= (\Phi'(x+t))^2 - (\Psi'(x-t))^2,
\end{align}
using the first-derivative formulas
$\partial_t[A(x+t)+B(x-t)] = A'(x+t) - B'(x-t)$ and
$\partial_x[\cdots] = A'(x+t) + B'(x-t)$. Differentiating the energy density and the flux and
comparing gives the conservation law with no analytic side conditions --- the mixed partials
are manifestly symmetric for a traveling wave.

Two corollaries complete the section: the energy density is nonnegative ($e \ge 0$, being
$\tfrac{1}{2}$ a sum of squares), and the local conservation law specializes to the
d'Alembert solution via the superposition decomposition $\Phi = (g+H)/2$, $\Psi = (g-H)/2$.
This provides the local conservation law and nonnegativity underlying the energy method.

\subsection{§2.4.3 Uniqueness via Riemann Invariants}

For the 1D Cauchy problem the solution is unique. The textbook energy argument integrates
$e$ over a backward characteristic cone; that route needs $\int$-energy conservation
(differentiation under the integral sign, the same machinery developed for the Heat chapter).
In one dimension, however, uniqueness follows from a clean \emph{pointwise} argument via the
Riemann invariants $u_t \pm u_x$, with no integration of the energy required.

\textbf{Zero data $\Rightarrow$ zero solution} (proved): for a traveling-wave solution
$A(x+t) + B(x-t)$, zero initial position ($A + B \equiv 0$) and zero initial velocity
($A' - B' \equiv 0$) force both Riemann invariants $A' + B'$ and $A' - B'$ to vanish.
Differentiating the position condition gives $A' + B' \equiv 0$; combined with the velocity
condition, $A' \equiv B' \equiv 0$, so $A$ and $B$ are constant (a function with identically
zero derivative is constant) and cancel by their values at $0$.

\textbf{Uniqueness} (proved): two traveling-wave solutions with the same initial position
and velocity coincide everywhere, by applying the previous result to the difference of their
profiles. Since every $C^2$ solution of the 1D wave equation is a traveling-wave
superposition, this is uniqueness for the Cauchy problem --- in particular, the d'Alembert
formula gives the unique such solution of its data.

\subsection{§2.4.3 Finite Propagation Speed}

\textbf{Domain of dependence} (proved): $\texttt{dalembert}\,g\,h\,(x_0, t_0)$ depends only
on the initial data $(g, h)$ restricted to the interval $[x_0 - t_0,\, x_0 + t_0]$. If two
pairs of data agree on that interval, the solutions agree at $(x_0, t_0)$. The proof reads
off the three places the formula samples the data --- the endpoints $x_0 \pm t_0$ for the
$g$-terms and the interval $[x_0-t_0, x_0+t_0]$ for the integral of $h$ --- all within the
dependence interval. Disturbances therefore propagate at speed at most $1$.

\subsection{§2.4.3 Symmetry of Mixed Partials and General Energy Conservation}

The energy and uniqueness results above are proved for the traveling-wave form
$\Phi(x+t)+\Psi(x-t)$, where the symmetry of mixed partials $\partial_t\partial_x u =
\partial_x\partial_t u$ is manifest. For a \emph{general} $C^2$ solution this symmetry is
the one extra ingredient (beyond $u_{tt}=u_{xx}$) needed to extend the energy conservation
law to all solutions. We prove it directly.

\textbf{Symmetry of mixed partials} (proved): for $u \in C^2(\mathbb{R}^2)$,
$\partial_t(\partial_x u) = \partial_x(\partial_t u)$. This is the classical equality of mixed
second partial derivatives of a twice continuously differentiable function (Clairaut/Schwarz),
obtained here by relating the one-variable slice derivatives to the second Fréchet derivative
of $u$, which is symmetric.

\textbf{General energy conservation} (proved): with this symmetry, the local conservation law
$\partial_t e = \partial_x p$ holds for \emph{every} $C^2$ solution, not just a traveling
wave. Differentiating the energy density and the flux directly, the difference reduces to
$u_t(u_{tt}-u_{xx}) + u_x(\partial_t\partial_x u - \partial_x\partial_t u)$; the first term
vanishes because $u$ solves the wave equation, and the second by the symmetry of mixed
partials. This is the general energy method of Evans §2.4.3, in full generality.

(Uniqueness is established for the traveling-wave class; extending it to all $C^2$ solutions
needs the general structure theorem for solutions of the 1D wave equation, left as future
work.)

\textbf{The entire Wave chapter is fully closed --- zero \texttt{sorry}.} It covers
existence, regularity, both initial conditions, the energy conservation law (for both the
traveling-wave form and \emph{all} $C^2$ solutions), uniqueness for the Cauchy problem,
finite propagation speed, and the symmetry of mixed partials: a complete formalization of
Evans §2.4.1 and §2.4.3 in one space dimension.

\section{Calculus Utilities: \texttt{Common/\allowbreak Calculus.\allowbreak lean}}
% ══════════════════════════════════════════════════════════════════════════════

\subsection{Motivation}

\texttt{Common/\allowbreak Calculus.\allowbreak lean} provides a shared library of spacetime calculus definitions
for functions $u : \mathbb{R}^n \times \mathbb{R} \to \mathbb{R}$, matching Evans'
notation throughout the project.

\subsection{Definitions}

\subsubsection{Spatial Gradient}

\begin{definition}
The \textbf{spatial gradient} $Du(x,t)$ is the gradient of the map
$x' \mapsto u(x', t)$ at $x$, viewed as a vector in $\mathbb{R}^n$.
\end{definition}

\begin{lstlisting}[caption={Spatial gradient in Lean}]
noncomputable def spatialGradient (u : Rn x R -> R) (p : Rn x R) : Rn :=
  gradient (fun x => u (x, p.2)) p.1
\end{lstlisting}

In Mathlib, \texttt{gradient f x} is defined for $f : F \to \mathbb{R}$ on an
inner product space $F$, and returns the unique vector $\nabla f(x) \in F$
satisfying $\langle \nabla f(x), v \rangle = \text{fderiv}_{\mathbb{R}}\, f\, x\, v$
for all $v$. Note: \texttt{gradient} does \emph{not} take $\mathbb{R}$ as an
explicit argument --- the field is inferred automatically from the function's type.

\subsubsection{Time Derivative}

\begin{definition}
The \textbf{time derivative} $u_t(x,t)$ is the derivative of the map
$t' \mapsto u(x, t')$ at $t$.
\end{definition}

\begin{lstlisting}[caption={Time derivative in Lean}]
noncomputable def timeDerivative (u : Rn x R -> R) (p : Rn x R) : R :=
  deriv (fun t => u (p.1, t)) p.2
\end{lstlisting}

Here \texttt{deriv} is Mathlib's scalar derivative for functions $\mathbb{R} \to \mathbb{R}$.
When $u$ is not differentiable, \texttt{deriv} returns $0$ by convention,
so no differentiability hypothesis is needed in the definition itself.

\subsubsection{Laplacian}

The Laplacian $\Delta u$ is \textbf{not} redefined in \texttt{Common/\allowbreak Calculus.\allowbreak lean}.
Instead, \texttt{PDE/\allowbreak Laplace.\allowbreak lean} uses Mathlib's \texttt{Laplacian.laplacian} directly
(opened as \texttt{Δ}). The \texttt{spatialLaplacian} wrapper is provided for
spacetime functions:

\begin{lstlisting}[caption={Spatial Laplacian for spacetime functions}]
noncomputable def spatialLaplacian (u : Rn x R -> R) (p : Rn x R) : R :=
  Laplacian.laplacian (fun x => u (x, p.2)) p.1
\end{lstlisting}

\subsection{Differentiation Under the Integral (Leibniz Rule)}

\texttt{Common/\allowbreak Calculus.\allowbreak lean} provides a fully proved \emph{combined} Leibniz rule for an integral
with a parameter-dependent integrand \emph{and} a variable upper limit:
\[
  \frac{d}{dt'}\int_0^{t'} H(t',s)\,ds\Big|_{t'=t}
    = H(t,t) + \int_0^t \partial_1 H(t,s)\,ds,
\]
for jointly continuous $H$ and $\partial_1 H$. Mathlib has the two ingredients (FTC and
differentiation under the integral on a \emph{fixed} interval) but not their combination, and
the naive route is blocked by the absence of ``continuous partial derivatives $\Rightarrow$
Fréchet differentiable'' on $\mathbb{R}^2$. We sidestep that with the decomposition
$F = P + B + C$ (FTC piece $+$ fixed-interval parametric piece $+$ a cross-term
$\int_t^{t'}(H(t',s)-H(t,s))\,ds$ that is $O((t'-t)^2)$, hence has derivative $0$). The
cross-term lemma \texttt{hasDerivAt\_crossTerm} is the genuinely new piece. This discharges
Step~2 of \texttt{duhamelFormula\_solves}.

\subsection{Shared Analysis Toolkit}

\texttt{Common/\allowbreak Calculus.\allowbreak lean} also collects the dimension- and chapter-agnostic analysis lemmas that
arose while differentiating convolutions and radial potentials, so Heat and Laplace share them
rather than each carrying a private copy.

\paragraph{$n$-dimensional Gaussian moments.}
Integrability over $\mathbb{R}^n$ of $\|z\|^k e^{-c\|z\|^2}$ for $k=0,1,2$, any $c>0$. Mathlib
has the base $n$-dim Gaussian and the $1$-D moments only; these are the standard dominating
functions for Gaussian/heat-kernel convolutions.

\paragraph{Real inner product as a bilinear CLM.}
\texttt{realInnerBiL}, \texttt{realInnerL}, with \texttt{norm\_realInnerBiL\_apply}. Mathlib's
\texttt{innerSL R} is \emph{conjugate}-linear, which obstructs \texttt{fderiv}; this is the
same map retyped as an honest $\mathbb{R}$-bilinear CLM.

\paragraph{Radial power calculus.}
\texttt{hasFDerivAt\_norm\_rpow\_of\_ne}, \texttt{laplacian\_norm\_rpow\_eq}
($\Delta(\|x\|^p) = p(n+p-2)\|x\|^{p-2}$, $x\neq 0$), and \texttt{laplacian\_norm\_sq}
($\Delta\|\cdot\|^2 = 2n$ everywhere --- the canonical strictly subharmonic perturbation used
in the maximum principle). The rpow Laplacian is \emph{why} $\|x\|^{2-n}$ and $\log\|x\|$ are
harmonic.

\paragraph{Radial integrability on the ball.}
\texttt{integrableOn\_unitBall\_radial} (reduce a radial ball integral to a 1-D radial integral
via Mathlib's \texttt{integrable\_fun\_norm\_addHaar}) and \texttt{integrableOn\_norm\_rpow\_unitBall}
($\|\cdot\|^p\in L^1(B(0,1))$ iff $p>-n$). These resolve the former $n$-dim-polar-coordinates
blocker for the Laplace fundamental solution.

\paragraph{Second-derivative / maximum-principle helpers.}
\texttt{norm\_smul\_clm\_le}, \texttt{norm\_comp\_apply\_le}; the eval-bridge
\texttt{fderiv\_fderiv\_apply} (second Fréchet derivative $\to$ nested scalar directional
derivatives); \texttt{laplacian\_integral\_eq} (Laplacian under an integral sign); and the
analytic core of the maximum principle, \texttt{deriv\_deriv\_nonpos\_of\_isLocalMax}
($g''(t)\le 0$ at a local max) and \texttt{laplacian\_nonpos\_of\_isLocalMax} ($\Delta f\le 0$
at an interior local max of a $C^2$ function).

\paragraph{Horizontal integration by parts (for the divergence theorem).}
The Gauss--Green development of Chapter~6 rests on a multivariate integration-by-parts identity
built here from scratch, since Mathlib has no ``$\int_{\mathbb{R}^{m+1}}\partial_{x_i}(fg)=0$'' in
usable form. The chain is: the compact-support fundamental theorem of calculus
\texttt{integral\_deriv\_eq\_zero} ($\int_{\mathbb{R}} f' = 0$ for $f\in C^1_c$); the coordinate-slice
calculus along $\mathrm{Fin.insertNth}\,i$ (\texttt{hasDerivAt\_insertNth},
\texttt{contDiff\_insertNth}, \texttt{continuous\_insertNth}, \texttt{isClosedEmbedding\_insertNth}),
which lets one differentiate and integrate in the single free coordinate $x_i$; the single-coordinate
Fubini lift \texttt{integral\_eq\_zero\_of\_forall\_insertNth\_integral\_zero} (a function whose
integral over every $x_i$-slice vanishes integrates to $0$); and finally the horizontal IBP itself in
three progressively packaged forms --- \texttt{integral\_horizontal\_ibp},
\texttt{integral\_horizontal\_ibp'} (rearranged), and \texttt{integral\_horizontal\_ibp\_euclidean}
(bridged to the \texttt{EuclideanSpace} base so it composes with $\operatorname{div}_E$). These
supply the $\partial_{x_i}$ (flux) half of the graph Gauss--Green theorem (\S6.3).

\section{Chapter 5: Sobolev Spaces (Evans §5.2)}
% ══════════════════════════════════════════════════════════════════════════════

Sobolev spaces are the \textbf{main target} of the project: all of modern PDE
theory is phrased in their language. This chapter (\texttt{Sobolev/\allowbreak Basic.\allowbreak lean}) lays the
analytic foundations --- test functions, weak derivatives, and the space
$W^{1,p}$ --- and proves the core results connecting classical and weak calculus and
establishing that $W^{1,p}$ is a Banach space. Every statement below is fully proved and
machine-checked; the proofs are given here in ordinary mathematical language, in the style
of Evans.

\subsection{§5.2.1 Test functions and weak derivatives}

\begin{definition}
A \textbf{test function} on an open set $U \subseteq \mathbb{R}^n$ is a function
$\phi \in C^\infty_c(U)$: smooth, compactly supported, with $\operatorname{tsupp}\phi
\subseteq U$.
\end{definition}

\begin{lstlisting}[caption={Test functions}]
def IsTestFunction (U : Set Rn) (phi : Rn -> R) : Prop :=
  ContDiff R inf phi /\ HasCompactSupport phi /\ tsupport phi subset U
\end{lstlisting}

We work with first-order weak derivatives along a fixed direction $e \in \mathbb{R}^n$,
which is all that $W^{1,p}$ needs. The direction-$e$ partial $\partial_e\phi$ is the
Fréchet derivative of $\phi$ evaluated at $e$.

\begin{definition}[Evans §5.2.1]
$v$ is the \textbf{weak derivative of $u$ in the direction $e$} on $U$ if
\[
  \int_{\mathbb{R}^n} u(x)\, \partial_e \phi(x)\, dx
  = -\int_{\mathbb{R}^n} v(x)\, \phi(x)\, dx
  \qquad \forall \phi \in C^\infty_c(U).
\]
\end{definition}

\begin{lstlisting}[caption={Weak directional derivative}]
def IsWeakDerivInDir (U : Set Rn) (e : Rn) (u v : Rn -> R) : Prop :=
  ForAll phi, IsTestFunction U phi ->
    integral x, u x * fderiv R phi x e = - integral x, v x * phi x
\end{lstlisting}

\begin{theorem}[Classical $\Rightarrow$ weak]
If $u \in C^1$, then its classical directional derivative $x \mapsto \partial_e u(x)$
is a weak derivative of $u$ in the direction $e$.
\end{theorem}

\begin{lstlisting}[caption={Classical derivative is the weak derivative (proved)}]
theorem isWeakDerivInDir_of_contDiff (U : Set Rn) (e : Rn) {u : Rn -> R}
    (hu : ContDiff R 1 u) :
    IsWeakDerivInDir U e u (fun x => fderiv R u x e)
\end{lstlisting}

\begin{proof}
This is integration by parts with no boundary term. Fix $\phi\in C^\infty_c(U)$. The
product $u\phi$ is $C^1$ and compactly supported (its support lies in
$\operatorname{tsupp}\phi$), so $\int_{\mathbb{R}^n}\partial_e(u\phi)\,dx=0$: the integral
of a directional derivative of a compactly supported $C^1$ function vanishes. By the
product rule $\partial_e(u\phi)=(\partial_e u)\,\phi+u\,\partial_e\phi$, and each of the
functions $u\,\partial_e\phi$ and $(\partial_e u)\,\phi$ is continuous and supported in the
compact set $\operatorname{tsupp}\phi$, hence integrable. Integrating the product-rule
identity therefore gives $\int u\,\partial_e\phi=-\int(\partial_e u)\,\phi$, i.e.\
$\partial_e u$ is the weak $e$-derivative of $u$.
\end{proof}

The weak derivative is $\mathbb{R}$-linear: if $v_1,v_2$ are weak $e$-derivatives of
$u_1,u_2$, then for scalars $\alpha,\beta$ the function $\alpha v_1+\beta v_2$ is a weak
$e$-derivative of $\alpha u_1+\beta u_2$ (and similarly for sums, differences and
negatives). This follows at once from the defining identity once one may split the
integral $\int(\alpha u_1+\beta u_2)\,\partial_e\phi=\alpha\int u_1\,\partial_e\phi+\beta
\int u_2\,\partial_e\phi$, which is legitimate because each integrand is a locally
integrable function times a compactly supported test weight, hence integrable.

Further properties from Evans §5.2.3 hold:

\begin{itemize}
  \item \textbf{Constants.} A constant is smooth with vanishing classical derivative, so by
    the classical-to-weak result above its weak $e$-derivative is $0$, in every direction.
  \item \textbf{Locality.} A weak $e$-derivative on $U$ is also a weak $e$-derivative on any
    $V\subseteq U$, since every test function supported in $V$ is in particular a test
    function supported in $U$; consequently $W^{1,p}(U)\subseteq W^{1,p}(V)$.
  \item \textbf{Invariance under a.e.\ equality.} The defining identity sees $u$ and $v$
    only through their integrals against test functions and their derivatives, so it is
    unchanged if either is altered on a null set; the relation therefore descends to $L^p$
    equivalence classes. On a general domain $U$ it is even insensitive to changes
    \emph{outside} $U$ --- a test function and its derivative vanish there --- so it descends
    to $L^p(U)$ classes. This is the prerequisite for $W^{1,p}$ being a well-defined normed
    space.
  \item \textbf{Product rule with a smooth function.} If $v$ is the weak $e$-derivative of
    $u$ and $\psi$ is smooth, then $\psi u$ has weak $e$-derivative
    $\psi v+(\partial_e\psi)\,u$. Indeed, for any test function $\phi$ the product
    $\psi\phi$ is again a test function; applying the defining identity for $u$ to $\psi\phi$
    and using the Leibniz rule
    $\partial_e(\psi\phi)=\psi\,\partial_e\phi+(\partial_e\psi)\,\phi$ gives the claim after
    rearranging. This is the workhorse behind cutoff and localisation arguments.
\end{itemize}

\begin{theorem}[Uniqueness a.e.]
If $v_1$ and $v_2$ are both weak $e$-derivatives of $u$ on the open set $U$, then
$v_1 = v_2$ almost everywhere on $U$.
\end{theorem}

\begin{lstlisting}[caption={Weak derivative unique a.e. (proved)}]
theorem isWeakDerivInDir_ae_unique {U : Set Rn} (hU : IsOpen U) {e : Rn}
    {u v1 v2 : Rn -> R}
    (hv1 : LocallyIntegrable v1) (hv2 : LocallyIntegrable v2)
    (h1 : IsWeakDerivInDir U e u v1) (h2 : IsWeakDerivInDir U e u v2) :
    ForAll-ae x, x in U -> v1 x = v2 x
\end{lstlisting}

\begin{proof}
Subtracting the two defining identities gives $\int (v_1-v_2)\,\phi=0$ for every test
function $\phi\in C^\infty_c(U)$. By the \textbf{fundamental lemma of the calculus of
variations}, a locally integrable function whose integral against every smooth compactly
supported test function vanishes must be zero almost everywhere; applied on $U$ this forces
$v_1=v_2$ almost everywhere on $U$.
\end{proof}

\subsection{§5.2.2 The space $W^{1,p}$}

\begin{definition}
$u \in W^{1,p}(U)$ if $u \in L^p(U)$ and in every coordinate direction $e_i$ there
is a weak derivative of $u$ that also lies in $L^p(U)$.
\end{definition}

\begin{lstlisting}[caption={Membership in W^{1,p}}]
structure MemW1p (U : Set Rn) (p : ENNReal) (u : Rn -> R) : Prop where
  memLp : MemLp u p (volume.restrict U)
  exists_weakDeriv : ForAll i : Fin n, Exists v,
    IsWeakDerivInDir U (EuclideanSpace.single i 1) u v
      /\ MemLp v p (volume.restrict U)
\end{lstlisting}

\begin{theorem}[Prototypical Sobolev function]
Every $u \in C^\infty_c$ lies in $W^{1,p}(U)$ for all $p$.
\end{theorem}

\begin{lstlisting}[caption={Smooth compact support implies W^{1,p} (proved)}]
theorem memW1p_of_contDiff_hasCompactSupport (U : Set Rn) (p : ENNReal)
    {u : Rn -> R} (hu : ContDiff R inf u) (hsupp : HasCompactSupport u) :
    MemW1p U p u
\end{lstlisting}

\begin{proof}
By the classical-to-weak bridge, each classical partial $\partial_{e_i}u$ is the weak
$e_i$-derivative of $u$. A continuous, compactly supported function lies in $L^p$ for every
$p$: it is bounded and supported on a set of finite measure, so $\int|u|^p<\infty$. This
applies to $u$ and to each $\partial_{e_i}u$ (again continuous with compact support).
Restricting to $U$ gives $u\in L^p(U)$ with all weak partials in $L^p(U)$, i.e.\
$u\in W^{1,p}(U)$.
\end{proof}

\subsection{Closedness under limits (towards completeness)}

The analytic cornerstone of completeness of $W^{1,p}$ is that the weak derivative is
\emph{closed} under convergence: one may pass to the limit in the defining
integration-by-parts identity. We prove this for $L^1$-on-compacts convergence.

\begin{theorem}[Closedness]
Suppose each $v_k$ is a weak $e$-derivative of $u_k$, and $u_k\to u$, $v_k\to v$ in
$L^1$ on every compact set. Then $v$ is a weak $e$-derivative of $u$.
\end{theorem}

\begin{lstlisting}[caption={Closedness under L1-on-compacts limits (proved)}]
theorem isWeakDerivInDir_of_tendsto_L1 {U : Set Rn} {e : Rn} {u v : Rn -> R}
    {uk vk : Nat -> Rn -> R}
    (hweak : ForAll k, IsWeakDerivInDir U e (uk k) (vk k))
    (hukloc : ForAll k, LocallyIntegrable (uk k)) (huloc : LocallyIntegrable u)
    (hvkloc : ForAll k, LocallyIntegrable (vk k)) (hvloc : LocallyIntegrable v)
    (hu : ForAll K, IsCompact K ->
      Tendsto (fun k => integral x in K, |uk k x - u x|) atTop (nhds 0))
    (hv : ForAll K, IsCompact K ->
      Tendsto (fun k => integral x in K, |vk k x - v x|) atTop (nhds 0)) :
    IsWeakDerivInDir U e u v
\end{lstlisting}

\begin{proof}
Fix a test function $\phi$; for each $k$ we have $\int u_k\,\partial_e\phi=-\int
v_k\,\phi$. The key observation is that integration against a fixed continuous, compactly
supported weight $g$ is continuous under $L^1$-on-compacts convergence: writing
$K=\operatorname{tsupp}g$ and $C=\sup|g|$,
\[
  \Bigl|\int (w_k-w)\,g\Bigr|
  \le \int_K |w_k-w|\,|g|
  \le C\int_K |w_k-w| \;\longrightarrow\; 0,
\]
so $\int w_k\,g\to\int w\,g$. Applying this with $g=\partial_e\phi$ and with $g=\phi$ gives
$\int u_k\,\partial_e\phi\to\int u\,\partial_e\phi$ and $\int v_k\,\phi\to\int v\,\phi$.
Letting $k\to\infty$ in $\int u_k\,\partial_e\phi=-\int v_k\,\phi$ and using uniqueness of
limits yields $\int u\,\partial_e\phi=-\int v\,\phi$, i.e.\ $v$ is the weak $e$-derivative of
$u$.
\end{proof}

The same closedness holds for genuine $L^p$ convergence ($1\le p<\infty$), which is the
form needed for completeness of $W^{1,p}$.

\begin{theorem}[Closedness, $L^p$ form]
If each $v_k$ is a weak $e$-derivative of $u_k$ and $u_k\to u$, $v_k\to v$ in $L^p$,
then $v$ is a weak $e$-derivative of $u$.
\end{theorem}

\begin{lstlisting}[caption={Closedness under Lp limits (proved)}]
theorem isWeakDerivInDir_of_tendsto_Lp {U : Set Rn} {e : Rn} {u v : Rn -> R}
    {uk vk : Nat -> Rn -> R} {p : ENNReal} (hp1 : 1 <= p) (hp_ne : p != Top)
    (hweak : ForAll k, IsWeakDerivInDir U e (uk k) (vk k))
    (... loc-integrability of uk, u, vk, v ...)
    (humem : ForAll k, MemLp (fun x => uk k x - u x) p volume)
    (hvmem : ForAll k, MemLp (fun x => vk k x - v x) p volume)
    (hucon : Tendsto (fun k => eLpNorm (fun x => uk k x - u x) p volume) atTop (nhds 0))
    (hvcon : Tendsto (fun k => eLpNorm (fun x => vk k x - v x) p volume) atTop (nhds 0)) :
    IsWeakDerivInDir U e u v
\end{lstlisting}

\begin{proof}
It suffices to reduce $L^p$ convergence to $L^1$-on-compacts convergence and apply the
previous theorem. On a set $K$ of finite measure, Hölder's inequality gives the
\textbf{Hölder bridge}
\[
  \|h\|_{L^1(K)} \le \|h\|_{L^p(K)}\,\mu(K)^{1-1/p}.
\]
Since $\|u_k-u\|_{L^p(K)}\le\|u_k-u\|_{L^p}\to0$ (and likewise for $v$), the right-hand side
tends to $0$, so $\int_K|u_k-u|\to0$ on every compact $K$. These are exactly the hypotheses
of the $L^1$-on-compacts closedness theorem.
\end{proof}

Cast in the language of $L^p$ \emph{spaces}, this says the weak-derivative relation is a
\textbf{closed graph}, which is precisely the property that makes $W^{1,p}$ a Banach space:
a closed subspace of a complete space is complete.

\begin{theorem}[Closed graph in $L^p\times L^p$]
For $1\le p<\infty$, the set of pairs $(f,g)\in L^p(\mathbb{R}^n)\times L^p(\mathbb{R}^n)$
such that $g$ is the weak $e$-derivative of $f$ is closed.
\end{theorem}

\begin{lstlisting}[caption={Weak-derivative graph is closed in Lp x Lp (proved)}]
theorem isClosed_isWeakDerivInDir_graph {p : ENNReal} [Fact (1 <= p)]
    (hp_ne : p != Top) (e : Rn) :
    IsClosed {fg : Lp R p volume x Lp R p volume |
      IsWeakDerivInDir univ e (coeFn fg.1) (coeFn fg.2)}
\end{lstlisting}

\begin{proof}
$L^p$ is a metric space, so it suffices to check sequential closedness. Suppose
$(f_k,g_k)\to(f,g)$ in $L^p\times L^p$ with each $g_k$ the weak $e$-derivative of $f_k$.
Convergence in $L^p\times L^p$ means $\|f_k-f\|_{L^p}\to0$ and $\|g_k-g\|_{L^p}\to0$, so by
the $L^p$ closedness theorem $g$ is the weak $e$-derivative of $f$; thus the limit
$(f,g)$ again lies in the set.
\end{proof}

\paragraph{A Banach space.}
Promoting the closed graph to a genuine subspace yields an honest Banach space. The
single-direction Sobolev space is the linear subspace
\[
  \{\,(f,g)\in L^p\times L^p : \text{$g$ is the weak $e$-derivative of $f$}\,\}
  \le L^p\times L^p,
\]
a subspace because the weak derivative is linear --- and the defining relation is
well-defined on $L^p$ equivalence classes, since it is insensitive to changes on null sets.

\begin{lstlisting}[caption={Single-direction Sobolev space is a Banach space (proved)}]
theorem completeSpace_weakDerivSubmodule {p : ENNReal} [Fact (1 <= p)]
    (hp_ne : p != Top) (e : Rn) :
    CompleteSpace (weakDerivSubmodule e)
\end{lstlisting}

\begin{proof}
The subspace is closed in $L^p\times L^p$ by the closed-graph theorem, and a closed linear
subspace of a complete normed space is itself complete. Hence it is a Banach space.
\end{proof}

\paragraph{The full space $W^{1,p}(\mathbb{R}^n)$.}
The genuine Sobolev space pairs $f$ with its full weak gradient $g=(g_1,\dots,g_n)$, each
$g_i$ the weak $\partial_i$-derivative of $f$ (i.e.\ the weak derivative in the $i$-th
coordinate direction):
\[
  \{(f,g)\in L^p\times (L^p)^n : \forall i,\ \text{$g_i$ is the weak
  $\partial_i$-derivative of $f$}\}.
\]
This is the intersection over $i$ of the single-direction graphs, pulled back along the
continuous coordinate projections $(f,g)\mapsto(f,g_i)$; as a finite intersection of closed
subspaces it is closed, hence complete.

\begin{lstlisting}[caption={W^{1,p}(Rn) is a Banach space (proved)}]
theorem completeSpace_weakGradientSubmodule {p : ENNReal} [Fact (1 <= p)]
    (hp_ne : p != Top) : CompleteSpace (weakGradientSubmodule (n := n) (p := p))
\end{lstlisting}

This is the headline result of the chapter: \textbf{$W^{1,p}(\mathbb{R}^n)$ is a Banach
space} for $1\le p<\infty$, built on the weak-derivative theory developed here together with
the completeness of $L^p$.

\paragraph{The Sobolev norm.}
The construction above gives a Banach space, but with the ambient product norm (a
$\max$/$\sup$ combination of the $L^p$ norms of the components). To carry the \emph{genuine}
Sobolev norm
\[
  \|u\|_{W^{1,p}} = \Bigl(\|u\|_p^p + \sum_{i=1}^n \|\partial_i u\|_p^p\Bigr)^{1/p}
\]
one instead places the tuple $(f,\partial_1 f,\dots,\partial_n f)$ in the $\ell^p$-product of
$n+1$ copies of $L^p$, whose norm is exactly this $\ell^p$ combination. All norm axioms ---
in particular the triangle inequality (Minkowski) --- are then inherited from the
$\ell^p$-product.

\begin{lstlisting}[caption={W^{1,p}(Rn) with its Sobolev norm is a Banach space (proved)}]
def sobolevSpace {p : ENNReal} [Fact (1 <= p)] :
    Submodule R (PiLp p (fun _ : Fin (n+1) => Lp R p volume))

lemma norm_eq_sobolev (hp_ne : p != Top)
    (x : PiLp p (fun _ : Fin (n+1) => Lp R p volume)) :
    ||x|| = (sum i, ||x i|| ^ p.toReal) ^ (1 / p.toReal)

theorem completeSpace_sobolevSpace (hp_ne : p != Top) :
    CompleteSpace (sobolevSpace (n := n) (p := p))
\end{lstlisting}

The closedness argument is the same (a finite intersection of single-direction graphs,
pulled back along the continuous coordinate projections $x\mapsto(x_0,x_{i+1})$), so
$(\,W^{1,p}(\mathbb{R}^n),\ \|\cdot\|_{W^{1,p}}\,)$ is a genuine Banach space --- now with the
textbook norm, whose axioms all come from the $\ell^p$-product structure.

\subsection{$W^{1,p}(U)$ on a general domain (restricted measure)}

Everything so far takes $U=\mathbb{R}^n$. For a general measurable set $U$ the genuine
$W^{1,p}(U)$ uses the \emph{restricted} measure $\mathrm{vol}\restriction U$ --- i.e.\ real
$L^p(U)$, whose elements are pinned down only almost everywhere on $U$. Three points make the
construction go through.

\begin{enumerate}
  \item \textbf{Well-definedness on $L^p(U)$ classes.} The weak-derivative relation on $U$
    depends on $u,v$ only up to almost-everywhere equality \emph{on $U$}: values off $U$ are
    invisible, because a test function $\phi$ and its derivative $\partial_e\phi$ vanish
    outside $\operatorname{tsupp}\phi\subseteq U$. Hence the relation descends to $L^p(U)$
    equivalence classes.
  \item \textbf{Bridging the measures.} The integrals defining the relation are taken over all
    of $\mathbb{R}^n$; but since $\phi$ and $\partial_e\phi$ vanish off $U$, the integrand
    vanishes there too, so these integrals equal the corresponding integrals over $U$. The
    convergence arguments above therefore apply verbatim with Lebesgue measure replaced by its
    restriction to $U$ --- indeed they were carried out for an arbitrary measure, $\mathbb{R}^n$
    being the case of full Lebesgue measure.
  \item \textbf{Localised algebra.} Additivity of the weak derivative needs only local
    integrability on $U$: a product of a function locally integrable on $U$ with a compactly
    supported test weight is globally integrable, being supported in $U$ (where the function is
    integrable) and zero elsewhere.
\end{enumerate}

\begin{lstlisting}[caption={W^{1,p}(U) over the restricted measure is a Banach space (proved)}]
theorem isWeakDerivInDir_of_tendsto_Lp_restrict {U : Set Rn} ... :
    -- Lp(U)-convergence closedness cornerstone
    IsWeakDerivInDir U e u v

theorem isClosed_isWeakDerivInDir_graph_restrict {U : Set Rn} {p : ENNReal}
    [Fact (1 <= p)] (hp_ne : p != Top) (e : Rn) :
    IsClosed {fg : Lp R p (volume.restrict U) x Lp R p (volume.restrict U) |
      IsWeakDerivInDir U e (coeFn fg.1) (coeFn fg.2)}

def weakGradientSubmoduleOn (U : Set Rn) (hU : MeasurableSet U) {p : ENNReal} [Fact (1 <= p)] :
    Submodule R (Lp R p (volume.restrict U) x (Fin n -> Lp R p (volume.restrict U)))

theorem completeSpace_weakGradientSubmoduleOn (U : Set Rn) (hU : MeasurableSet U)
    {p : ENNReal} [Fact (1 <= p)] (hp_ne : p != Top) :
    CompleteSpace (weakGradientSubmoduleOn U hU (n := n) (p := p))
\end{lstlisting}

The closed-graph argument is the same as on $\mathbb{R}^n$ but uses the restricted-measure
form of closedness; the subspace axioms follow from linearity of the weak derivative together
with its invariance under a.e.-equality on $U$. Thus \textbf{$W^{1,p}(U)$ is a Banach space} for
every measurable $U$ and $1\le p<\infty$ --- the genuine, textbook Sobolev space on a general
domain.

\subsection{A bundled interface}

For downstream use the space is packaged as a named type with the usual operators. We expose
$W^{1,p}(U)$ as an abbreviation \texttt{W1p U hU p} for the underlying space of the weak-gradient
subspace; it inherits the normed-space structure from the ambient product, and completeness is
registered as a typeclass instance (for $1\le p$ and $p\ne\infty$, the latter recorded as a
\texttt{Fact}), so $W^{1,p}(U)$ is usable as a Banach space without re-supplying the proof each
time.

The two natural maps out of the space are then bounded linear operators. The function-value map
$u=(f,Df)\mapsto f$ and, for each coordinate $i$, the weak partial derivative $u\mapsto D_iu$ are
each obtained by composing the continuous inclusion of the closed subspace into the ambient
product with a coordinate projection; composites of continuous linear maps are continuous, so both
are honest elements of the operator space $W^{1,p}(U)\to L^p(U)$. This realises the abstract weak
gradient as a bounded operator, the form in which it is used in the rest of the theory.

\begin{lstlisting}[caption={W^{1,p}(U) as a bundled Banach space with bounded operators (proved)}]
abbrev W1p (U : Set Rn) (hU : MeasurableSet U) (p : ENNReal) [Fact (1 <= p)] : Type _ :=
    weakGradientSubmoduleOn U hU (n := n) (p := p)

instance instCompleteSpaceW1p {U : Set Rn} {hU : MeasurableSet U} {p : ENNReal}
    [Fact (1 <= p)] [Fact (p != Top)] : CompleteSpace (W1p U hU p)

def W1p.toLpCLM (U : Set Rn) (hU : MeasurableSet U) {p : ENNReal} [Fact (1 <= p)] :
    W1p U hU p ->L[R] Lp R p (volume.restrict U)        -- (f, Df) |-> f

def W1p.weakDerivCLM (U : Set Rn) (hU : MeasurableSet U) {p : ENNReal} [Fact (1 <= p)]
    (i : Fin n) : W1p U hU p ->L[R] Lp R p (volume.restrict U)   -- (f, Df) |-> D_i f
\end{lstlisting}

\subsection{Mathlib foundations leveraged}

Three Mathlib results --- which a preliminary survey expected to be missing --- made
this chapter possible without a sorry:
\begin{itemize}
  \item \texttt{integral\_mul\_fderiv\_eq\_neg\_fderiv\_mul\_of\_integrable}
    (integration by parts in a direction, integrable form) --- the
    classical-to-weak bridge.
  \item \texttt{IsOpen.ae\_eq\_zero\_of\_integral\_contDiff\_smul\_eq\_zero}
    (fundamental lemma of the calculus of variations on an open set) --- uniqueness.
  \item \texttt{LocallyIntegrable.integrable\_smul\_*\_of\_hasCompactSupport} and
    \texttt{Continuous.memLp\_of\_hasCompactSupport} --- the integrability and
    $L^p$-membership plumbing.
\end{itemize}

\begin{remark}[Implementation pitfall]
The test-function smoothness must be $C^\infty$ (\texttt{ContDiff R inf}, i.e.\
$\infty : \mathtt{With\-Top}\ \mathbb{N}_\infty$), \emph{not} $\top$, which in
current Mathlib denotes the strictly stronger \emph{analytic} class $\omega$. A
separate trap: introducing the point/function arguments via a section
\texttt{variable} whose type unfolds through the \texttt{Rn} notation silently
elaborates the declaration with a \texttt{sorry}; binding those arguments
explicitly in each lemma avoids it.
\end{remark}

\subsection{Mollification and the Meyers--Serrin theorem (§5.3)}

This section builds the \emph{mollification} layer and uses it to prove the Meyers--Serrin
theorem (`$H=W$'): smooth functions approximate Sobolev functions, together with their weak
derivatives, in $L^p$.  Throughout, $\eta_\delta$ denotes a normalized bump function (smooth,
nonnegative, $\int\eta_\delta = 1$, supported in the ball of radius $\delta$) and $\eta_\delta\star u$
its convolution with $u$.  The development fills three gaps in Mathlib --- continuity of
translation in $L^p$, a weighted Jensen inequality, and $L^p$-convergence of mollification ---
none of which were available.

\paragraph{Continuity of translation in $L^p$.}
For $u\in L^p$ ($1\le p<\infty$), the translates $u(\cdot+t)$ depend continuously on $t$:
$\|u(\cdot+t)-u\|_p \to 0$ as $t\to 0$.  The proof is the classical three-$\varepsilon$ argument.
Given $\varepsilon$, approximate $u$ within $\varepsilon/3$ in $L^p$ by a continuous, compactly
supported $g$ (these are dense).  Translation is an $L^p$-isometry, so $u(\cdot+t)$ and $g(\cdot+t)$
also differ by at most $\varepsilon/3$.  Finally $g$ is uniformly continuous and supported in a fixed
compact set once $\|t\|\le1$, so $\|g(\cdot+t)-g\|_p$ is at most $(\operatorname{vol} L)^{1/p}$ times
the uniform modulus of continuity, which is below $\varepsilon/3$ for small $t$.  The triangle
inequality combines the three bounds.

\paragraph{$L^p$-convergence of mollification.}
The key estimate is
\[
  \|\eta_\delta\star u - u\|_p^p \;\le\; \int \eta_\delta(y)\,\|u(\cdot - y) - u\|_p^p \, dy .
\]
Because $\int\eta_\delta = 1$, one writes $(\eta_\delta\star u)(x) - u(x) = \int \eta_\delta(y)\,(u(x-y)-u(x))\,dy$,
applies the triangle inequality and then a \emph{weighted Jensen inequality} (the probability
weight $\eta_\delta$, exponent $p\ge1$, proved here in $\mathbb{R}_{\ge0}^\infty$ from Hölder's
inequality) to move the $p$-th power inside, and finally swaps the order of integration by Tonelli.
Since the integrand on the right is supported where $\|y\|\le\delta$ and the inner norm tends to $0$
as $y\to0$ by the previous paragraph, the right-hand side tends to $0$.  Hence
$\eta_\delta\star u \to u$ in $L^p$ as $\delta\to0$.

\paragraph{The mollified weak derivative.}
If $v$ is the weak derivative of $u$ in direction $e$, then $\eta_\delta\star u$ is smooth and its
(classical $=$ weak) derivative is $\eta_\delta\star v$.  Equivalently, $\eta_\delta\star u$ has weak
derivative $\eta_\delta\star v$.  This is proved directly from the definition of the weak derivative:
pairing $\eta_\delta\star u$ against a test function $\psi$ and unfolding the convolution gives a double
integral; Fubini (justified by integrability of $\eta_\delta(t)\,u(x-t)\,\partial_e\psi(x)$, which has
compact support in both variables) exchanges the order, and after a translation the inner integral is
exactly the weak-derivative identity for $u$ tested against the translate $\psi(\cdot+t)$.  Carrying the
resulting sign back through Fubini identifies the pairing as $-\int(\eta_\delta\star v)\,\psi$, which is
the weak-derivative relation for $\eta_\delta\star u$.

\paragraph{Meyers--Serrin ($H=W$).}
Combining the three results: for $u\in L^p$ with weak derivative $v\in L^p$, the mollification
$w=\eta_\delta\star u$ is smooth, has weak derivative $w'=\eta_\delta\star v$, and as $\delta\to0$ both
$\|u-w\|_p\to0$ and $\|v-w'\|_p\to0$.  Choosing $\delta$ small enough makes both errors below any
prescribed $\varepsilon$, so $(u,v)$ is approximated in $L^p\times L^p$ by a smooth function and its
weak derivative.

\subsection{Sobolev embedding (Gagliardo--Nirenberg--Sobolev)}

For a continuously differentiable, compactly supported $u$ on $\mathbb{R}^n$ and $1\le p<n$, the
$L^{p^*}$ norm is controlled by the $L^p$ norm of the gradient,
\[
  \|u\|_{p^*} \;\le\; C\,\|Du\|_p, \qquad \tfrac{1}{p^*} = \tfrac1p - \tfrac1n ,
\]
and more generally $\|u\|_q \le C\,\|Du\|_p$ for every $q$ in the Sobolev range
$\tfrac1p-\tfrac1n\le\tfrac1q$.  This is Mathlib's Gagliardo--Nirenberg--Sobolev inequality
specialized to the Euclidean ambient space (rewriting $\operatorname{finrank}\mathbb{R}^n = n$);
since a $C^1$ function's weak derivative coincides with its Fréchet derivative
(\texttt{isWeakDerivInDir\_of\_contDiff}), the right-hand side is exactly the $W^{1,p}$ gradient
seminorm of $u$.

The embedding \emph{passes to the limit} on all of $W^{1,p}(\mathbb{R}^n)$.  If $u$ is the
$W^{1,p}$-limit of $C^1$ compactly supported functions $u_k$ --- $u_k\to u$ in $L^p$ and
$Du_k\to V$ in $L^p$ --- then $u$ inherits the bound $\|u\|_{p^*}\le C\|V\|_p$ with the \emph{same}
(uniform) constant (\texttt{exists\_eLpNorm\_le\_eLpNorm\_fderiv\_of\_tendsto}).  The argument is
the standard density passage: the GNS constant is uniform across the sequence; $L^p$-convergence
yields an a.e.\ convergent subsequence (convergence in measure,
\texttt{tendstoInMeasure\_of\_tendsto\_eLpNorm} then
\texttt{TendstoInMeasure.exists\_seq\_tendsto\_ae}); Fatou lower-semicontinuity of the seminorm
(\texttt{Lp.eLpNorm'\_lim\_le\_liminf\_eLpNorm'}) passes the inequality to the limit; and the
right-hand side converges since $\|Du_k\|_p\to\|V\|_p$ (norm-continuity in $L^p$).  The required
approximating sequence is supplied by the now-proved \textbf{density of $C^\infty_c$ in
$W^{1,p}(\mathbb{R}^n)$}, \texttt{exists\_contDiff\_hasCompactSupport\_forall\_isWeakDerivInDir}:
truncate by a scaled cutoff (\texttt{exists\_cutoff\_family}) to compact support --- the two $L^p$
truncation limits \texttt{tendsto\_eLpNorm\_cutoff\_mul\_sub} and
\texttt{tendsto\_eLpNorm\_fderiv\_cutoff\_mul} (with the weak Leibniz rule
\texttt{IsWeakDerivInDir.mul\_smooth}) --- then mollify keeping compact support
(\texttt{HasCompactSupport.convolution}), glued by $\varepsilon/2+\varepsilon/2$.  Finally the
\textbf{embedding holds with no hypotheses} (\texttt{exists\_eLpNorm\_le\_eLpNorm\_fderiv\_of\_forall\_isWeakDerivInDir}):
if $u\in L^p$ has weak derivatives given by a CLM-valued $V$ ($\partial_{e_i}u=V\cdot e_i$, $V\in L^p$),
then $\|u\|_{p^*}\le C\|V\|_p$.  The per-direction convergence is turned into
$\|\mathrm{fderiv}\,u_k-V\|_p\to0$ by the \emph{components bridge} $\|L\|\le\sum_i\|L\,e_i\|$
(\texttt{opNorm\_le\_sum\_apply\_single}, lifted to $L^p$ by \texttt{eLpNorm\_clm\_sub\_le\_sum}); for
smooth $u_k$ the weak partials agree a.e.\ with \texttt{fderiv} (\texttt{isWeakDerivInDir\_ae\_unique}).
The bridge's $\mathbb{R}^n$ inner-product instance diamond (which blocks \texttt{rw}) was dodged by
Riesz representation and closing with \texttt{exact} (definitional, not syntactic) equality.

\subsection{Poincar\'e's inequality (§5.6)}

For a continuously differentiable $u$ supported in a bounded set and $1\le p<n$, the $L^p$ norm of
$u$ itself is controlled by the $L^p$ norm of its gradient,
\[
  \|u\|_p \;\le\; C\,\|Du\|_p ,
\]
the $W^{1,p}_0$ form of Poincar\'e's inequality (Evans §5.6, Theorem~1, case $q=p$).  This is the
specialization $q=p$ of the Sobolev--Poincar\'e estimate of the previous subsection: the subcritical
condition $\tfrac1p-\tfrac1n\le\tfrac1q$ holds trivially for $q=p$ since $\tfrac1n\ge0$.  The full
subcritical range $\|u\|_q\le C\|Du\|_p$ for $1\le q\le p^*$ is exactly the embedding lemma
\texttt{\_of\_le} already proved.

construction.  \emph{Symmetry of mixed weak partials} (Clairaut/Schwarz for weak derivatives) is

derivative) and the fundamental lemma.  Finally the \textbf{Hilbert space}

\subsection{Rellich--Kondrachov compactness (§5.7)}

The \emph{sufficient direction} of Rellich--Kondrachov is formalized end to end across
\texttt{Sobolev/\allowbreak Rellich.\allowbreak lean}, \texttt{Sobolev/\allowbreak FrechetKolmogorov.\allowbreak lean} and
\texttt{Sobolev/\allowbreak RellichKondrachov.\allowbreak lean} --- this is the $L^p$-compactness criterion that Mathlib
lacks.  The named theorem \texttt{rellich\_kondrachov}: a family of $C^1$ functions with a uniform
$L^p$ gradient bound $\|Du_i\|_p\le M$ (and a uniform $L^p$ bound) is \emph{totally bounded in}
$L^p(K)$ on a compact $K$ --- the compactness behind
$W^{1,p}(U)\hookrightarrow\hookrightarrow L^p(U)$.  It is built from three pieces, none of which
Mathlib provides:
\begin{itemize}
  \item the \textbf{translation/gradient estimate} $\|u(\cdot+h)-u\|_p\le\|h\|\,\|Du\|_p$
    (\texttt{eLpNorm\_translate\_sub\_le\_fderiv}, from the segment FTC
    \texttt{sub\_eq\_integral\_fderiv\_segment} by weighted Jensen, Tonelli and
    translation-invariance; packaged as \texttt{eLpNorm\_translate\_sub\_le\_of\_gradient\_le} with
    vanishing modulus \texttt{tendsto\_enorm\_mul\_nhds\_zero}), which supplies the uniform
    equicontinuity;
  \item the \textbf{Fréchet--Kolmogorov $L^p$-precompactness criterion}: the closed-embedding
    Arzelà--Ascoli step (range $\{\text{Continuous}\}$ closed in the uniform-on-compacts topology)
    into Mathlib's relative-compactness Arzelà--Ascoli
    (\texttt{isCompact\_closure\_toLp\_image\_of\_equicontinuous\_of\_bound}), the continuous-linear
    $C(K)\to L^p$ transfer, the measure-preserving isometry
    $L^p(\hat K,\mathrm{comap})\cong L^p(K,\mathrm{restrict})$
    (\texttt{continuousMap\_toLp\_comap\_eq\_compMeasurePreserving},
    \texttt{isCompact\_closure\_of\_isometry}), the convolution dischargers, the per-$\delta$
    precompactness \texttt{isCompact\_closure\_toLp\_restrict\_convolution}, the $\varepsilon/3$ glue
    \texttt{totallyBounded\_of\_forall\_approx}, and the mollifiable-form criterion
    \texttt{totallyBounded\_toLp\_restrict\_of\_mollifiable};
  \item the \textbf{uniform mollification bound}
    \texttt{eLpNorm\_convolution\_sub\_le\_of\_modulus} (integral form via the convolution bridge
    \texttt{convolution\_eq\_integral\_sub}).
\end{itemize}
These assemble into the \emph{self-contained} criterion
\texttt{totallyBounded\_toLp\_restrict\_of\_equicontinuous} (discharging mollifiability with an
explicit normalised \texttt{ContDiffBump}), whence \texttt{rellich\_kondrachov}.

The \emph{necessary} direction (the converse) is also formalized, in its equicontinuity form:
\texttt{uniformEquicontinuous\_translate\_of\_totallyBounded} shows that a family totally bounded in
$L^p(\mathbb{R}^n)$ is automatically uniformly $L^p$-equicontinuous,
$\sup_i\|u_i(\cdot-y)-u_i\|_p\to0$ as $y\to0$ --- exactly the equicontinuity hypothesis consumed by
the sufficient direction.  The proof is the symmetric $\varepsilon/3$ argument: a finite
$\varepsilon/3$-net of the (totally bounded) image, each net point being $L^p$-translation-continuous
(\texttt{tendsto\_eLpNorm\_translate\_sub}) uniformly over the finite net via
\texttt{Set.Finite.eventually\_all}, and the splitting of $u_i(\cdot-y)-u_i$ through a nearby net
point, two of whose three terms are $\|u_i-z\|_p<\varepsilon/3$ (translation is an $L^p$ isometry).
Only the tightness half (uniform smallness of mass at infinity), needed for the full whole-space
equivalence, is left.

\subsection{Future work}

The higher-order theory ($W^{k,p}$, $H^k$), completeness of $W^{1,p}$, the Meyers--Serrin density
theorem, the Gagliardo--Nirenberg--Sobolev embedding (now on \emph{all} of $W^{1,p}(\mathbb{R}^n)$,
with no smoothness or compact-support hypothesis, via $C^\infty_c$ density --- see the embedding
subsection) and Poincar\'e inequality, and the Rellich--Kondrachov compactness theorem are all
\emph{done} and documented in the subsections above.  What remains:

\begin{enumerate}
  \item \textbf{Poincar\'e--Wirtinger (mean-zero) inequality} $\|u-(u)_U\|_p\le C\|Du\|_p$, which
    rests on the Rellich--Kondrachov compactness now available.
  \item \textbf{Necessary direction of Fréchet--Kolmogorov}: the equicontinuity half is \emph{done}
    (\texttt{uniformEquicontinuous\_translate\_of\_totallyBounded}; see the Rellich--Kondrachov
    subsection); the remaining \emph{tightness} half --- a totally bounded family in $L^p(\mathbb{R}^n)$
    is uniformly $p$-tight at infinity --- would complete the full whole-space equivalence.
  \item \textbf{Bounded-domain Rellich--Kondrachov}: transfer the compact-$K$ statement to
    $W^{1,p}(U)\hookrightarrow\hookrightarrow L^p(U)$ on a bounded extension domain $U$ via a Sobolev
    extension operator.
  \item \textbf{Trace theorem} $u\mapsto u|_{\partial\Omega}$, $W^{1,p}(\Omega)\to L^p(\partial\Omega)$.
    The key estimate $\int_{\partial\Omega}|u|^p\,d\mu_H\le C\,\|u\|_{W^{1,p}(\Omega)}^p$ is a direct
    application of the Chapter~6 \texttt{divergence\_theorem} to $|u|^p$ times a vector field
    transverse to $\partial\Omega$, so the theorem is now \emph{unblocked}; what remains to build is
    the trace operator itself and the $C^1$-density passage.
  \item \textbf{Morrey} inequality ($W^{1,p}\hookrightarrow C^{0,\alpha}$ for $p>n$).
\end{enumerate}

\section{Chapter 6: Surface Measure and the General Divergence Theorem}
% ══════════════════════════════════════════════════════════════════════════════

Everything in this chapter lives in \texttt{Common/\allowbreak AreaFormula.\allowbreak lean}.  It is a self-contained
multi-week development whose endpoint is the \textbf{general Gauss--Green (divergence) theorem}
on a bounded $C^1$ domain --- a result Mathlib does not have (its only divergence theorem is on a
rectangular box, \texttt{MeasureTheory.integral\_divergence\_of\_hasFDerivAt\_off\_countable}).

\begin{remark}[Why this is the linchpin]
A survey of Evans' text shows the same missing ingredient behind the Sobolev \emph{trace} theorem,
the \emph{extension} operator, the bounded-domain \emph{Rellich--Kondrachov} and
\emph{Poincar\'e--Wirtinger} inequalities, and the classical \emph{representation formula} for the
Laplacian: all of them integrate against surface measure on a curved boundary and use Gauss--Green
there.  Mathlib has no surface measure on a $C^1$ hypersurface and no manifold Stokes theorem, so
this had to be built from scratch.  The chosen route --- the \emph{area formula} --- is the most
tractable door: it produces the surface measure directly as a pushforward of Lebesgue measure and
never needs the theory of integration on manifolds.
\end{remark}

Throughout, $\mathbb{R}^k$ abbreviates \texttt{EuclideanSpace R (Fin k)} (the listings write
\texttt{R\^{}k}), $\langle\cdot,\cdot\rangle$ is the real inner product, and the surface measure is the
dimension-normalized Euclidean Hausdorff measure $\mu_H^{[d]}$
(\texttt{Measure.euclideanHausdorffMeasure}, written \texttt{muHE[d]}), which coincides with
Lebesgue \texttt{volume} on any $d$-dimensional inner-product space.

\subsection{6.0 The final statement}

\begin{theorem}[General divergence theorem]
Let $\Omega\subseteq\mathbb{R}^{m+2}$ be a bounded $C^1$ domain, $\nu$ an outward unit normal
field on it, and $F:\mathbb{R}^{m+2}\to\mathbb{R}^{m+2}$ of class $C^1$.  Then
\[
  \int_{\Omega}\operatorname{div}F\,dx \;=\; \int_{\partial\Omega}\langle F,\nu\rangle\,d\mu_H .
\]
\end{theorem}

    (hOm : IsBoundedC1Domain Om) {nu : R^(m+2) -> R^(m+2)}
    (hnu : IsOutwardNormal Om nu) {F : R^(m+2) -> R^(m+2)} (hF : ContDiff R 1 F) :
    ∫ x in Om, divergenceE F x
      = ∫ x in frontier Om, <F x, nu x>_R ∂(muHE[m+1] : Measure (R^(m+2)))
\end{lstlisting}

The three hypotheses are the whole story: the domain is bounded $C^1$, $\nu$ is an outward normal,
and $F$ is $C^1$.  No sign conditions, no compact support (the partition of unity supplies it), no
finiteness assumption on the surface measure (it is a theorem).  The ambient dimension is written
$m+2$ so that the boundary dimension $m+1$ is a successor, which the graph machinery needs.

\begin{remark}[Scope --- what is and is not gained, and what a caller must supply]
The result is deliberately strong where it counts, and it is worth being explicit about both sides.
\emph{Strengths.} (i) The regularity demand is minimal: $F$ need only be $C^1$ (not $C^2$), and it
is \emph{not} required to be compactly supported --- the partition of unity manufactures that
internally (\S6.10). (ii) It holds on \emph{any} bounded $C^1$ domain, not a special shape; in
particular a ball or an annulus qualifies, which is exactly what the Laplace representation formula
(Chapter~2) needs, and where Mathlib offers only the rectangular box. (iii) It is
\emph{unconditional}: the earlier development carried a side hypothesis $\mu_H(\partial\Omega)<\infty$,
which \S6.11 discharges as a theorem, so no finiteness assumption is imposed on the caller.
\emph{Caveats a caller must know.} (a) The outward normal $\nu$ is an \emph{input}, not an output:
the theorem \emph{consumes} a field satisfying \texttt{IsOutwardNormal} but does \emph{not} construct
one or prove that a continuous outward normal exists on a general $C^1$ domain. A downstream user
(e.g.\ for a ball/annulus) must exhibit $\nu$ and verify \texttt{IsOutwardNormal} against the charts
--- this is precisely the ``plumbing'' referred to in the Chapter~2 status notes. (b) The
\texttt{IsBoundedC1Domain} charts straighten the boundary by a global \emph{rigid} change of frame
(a \texttt{LinearIsometryEquiv} $e$) followed by reading it as a graph --- the standard ``$C^1$
graph after rotation'' notion, not an arbitrary curvilinear chart. (c) The divergence on the
statement is the canonical trace divergence \texttt{divergenceE} on \texttt{EuclideanSpace}; the
sup-normed internal \texttt{divergence} of the graph computation never appears in the public API.
\end{remark}

The remainder of the chapter is the proof, built strictly bottom-up.  The dependency chain is
\[
  \text{area formula}\to\text{flux identity}\to\underbrace{\text{graph Gauss--Green}}_{\text{one local model}}
  \to\text{flatten}\to\text{bottomless}\to\text{chart flux}\to\text{chart term}\to\text{PoU assembly}.
\]

\subsection{6.1 The canonical divergence and its invariance}

The public divergence is the coordinate-free one on flat Euclidean space,
\[
  \operatorname{div}_E F(x)\;=\;\sum_i \big(\partial F(x)\,e_i\big)_i\;=\;\operatorname{tr}\!\big(DF(x)\big),
\]
where $e_i=$ \texttt{EuclideanSpace.single i 1}.  Writing it as a trace makes rotation/reflection
invariance a one-line consequence of $\operatorname{tr}(e^{-1}Ae)=\operatorname{tr}A$; this is the
property that lets a rotated chart be straightened.

\begin{lstlisting}[caption={Canonical divergence, trace form, and invariance}]
noncomputable def divergenceE {n : N} (F : R^n -> R^n) (x : R^n) : R :=
  ∑ i, fderiv R F x (EuclideanSpace.single i 1) i

theorem divergenceE_eq_trace (F) (x) : divergenceE F x = LinearMap.trace R _ (fderiv R F x)
theorem divergenceE_comp_isometry (e : R^n =li R^n) (hF : Differentiable R F) (x) :
    divergenceE (fun y => e.symm (F (e y))) x = divergenceE F (e x)
\end{lstlisting}

A second, \emph{internal} divergence \texttt{divergence} is used on the plain product
$\mathbb{R}^{m+1}\times\mathbb{R}$ during the graph computation.  That space carries the sup norm and
is \emph{not} an inner-product space, so it must never leak into the public API; the isometry
\texttt{flatten} (\S6.4) converts between the two.  \texttt{divergenceE\_finset\_sum} (additivity of
$\operatorname{div}_E$ over finite sums) is the linearity brick for the partition of unity, and
\texttt{integral\_divergenceE\_eq\_zero} ($\int_{\mathbb{R}^{m+1}}\operatorname{div}_E F=0$ for
compactly-supported $F$, via Fubini and the compact-support FTC) is the interior case of the
assembly.

\subsection{6.2 Surface measure via the area formula}

The surface measure is manufactured by the \textbf{area formula}: for a $C^1$ immersion
$\varphi:\mathbb{R}^m\to F$ injective on a measurable $A$,
\[
  \mu_H^{[m]}\big(\varphi(A)\big)\;=\;\int_A \operatorname{jac}(D\varphi)\,dx,
  \qquad \operatorname{jac}(M)=\sqrt{\det(M^{\!\top}M)} .
\]
The proof (documented in the project's area-formula notes) covers $A$ into near-linear cells via the
inverse function theorem, applies the linear image formula
$\mu_H^{[m]}(L''A)=\sqrt{\det(L^{\!\top}L)}\cdot\operatorname{vol}A$ on each cell after corestricting
to $\operatorname{range}L$, and passes to the limit.  The graph specialization and its
change-of-variables corollary are what the divergence theorem consumes:

\begin{lstlisting}[caption={Area formula (graph form) and vector-valued change of variables}]
theorem area_formula_graph {g : R^m -> R} (hg : ContDiff R 1 g) {A} (hA : MeasurableSet A) :
    (muHE[m] : Measure (WithLp 2 (R^m x R))) (graphFun g '' A)
      = ∫^- x in A, ENNReal.ofReal (Real.sqrt (1 + ‖gradient g x‖^2)) ∂volume

theorem setIntegral_image_graph_smul ... :
    ∫ y in graphFun g '' A, G y ∂muHE = ∫ x in A, Real.sqrt (1+‖gradient g x‖^2) • G (graphFun g x)
\end{lstlisting}

Here \texttt{graphFun g y = toLp (y, g y)} is the graph map into $\mathrm{WithLp}\,2\,
(\mathbb{R}^m\times\mathbb{R})$, so that the codomain is a genuine $(m{+}1)$-dimensional
inner-product space carrying $\mu_H^{[m]}$.  Notice the area formula is an \emph{ENNReal lower
integral} ($\int^-$); this exact shape is reused verbatim in \S6.11 to bound the boundary's surface
measure.

\subsection{6.3 The flux identity and the graph Gauss--Green theorem}

The upward unit normal to the graph of $\gamma$ is
$\nu_\gamma(x)=(-\nabla\gamma(x),1)/\sqrt{1+\|\nabla\gamma(x)\|^2}$.  Feeding it through the graph
change of variables makes the $\sqrt{1+\|\nabla\gamma\|^2}$ from the surface measure cancel the one
in the denominator of $\nu_\gamma$, producing the clean \textbf{flux identity}
\[
  \int_{\operatorname{graph}\gamma}\langle V,\nu_\gamma\rangle\,d\mu_H
    \;=\;\int_A\Big(V_2(x,\gamma x)-\big\langle V_1(x,\gamma x),\nabla\gamma(x)\big\rangle\Big)\,dx .
\]

\begin{lstlisting}[caption={Graph normal and the graph Gauss--Green theorem}]
def graphNormal (gamma : R^m -> R) (x : R^m) : WithLp 2 (R^m x R) :=
  (Real.sqrt (1 + ‖gradient gamma x‖^2))⁻¹ • toLp (-gradient gamma x, 1)

theorem divergence_theorem_graph {m : N} {gamma : R^(m+1) -> R} (hg : ContDiff R 1 gamma)
    {F : (R^(m+1) x R) -> (R^(m+1) x R)} (hF : ContDiff R 1 F) (hsupp : HasCompactSupport F) :
    (∫ x, ∫ t in (0:R)..(gamma x), divergence F (x, t))
      = (∫ y in graphFun gamma '' univ, <toLp (F y.ofLp), graphNormal gamma y.ofLp.1>_R ∂muHE[m+1])
          - ∫ x, (F (x, 0)).2
\end{lstlisting}

This is the load-bearing \emph{local model}: the divergence theorem for the region between the plane
$t=0$ and the graph $t=\gamma(x)$.  Its proof combines the vertical fundamental theorem of calculus
(the $\partial_t$ part gives the two boundary terms) with a multivariate horizontal integration by
parts (the $\partial_{x_i}$ parts give the $\langle\cdot,\nabla\gamma\rangle$ flux term).  Crucially
the vertical integral is a \emph{signed} interval integral $\int_0^{\gamma x}$, so
\textbf{no hypothesis $\gamma\ge 0$ is needed} --- a point that becomes decisive in \S6.5.

\subsection{6.4 Flattening to canonical coordinates}

The graph theorem speaks about the plain product $\mathbb{R}^{m+1}\times\mathbb{R}$; the public
theorem must speak about flat $\mathbb{R}^{m+2}$ with $\operatorname{div}_E$.  The bridge is the
volume- and surface-measure-preserving linear isometry
\[
  \texttt{flatten } m : \mathrm{WithLp}\,2\,(\mathbb{R}^{m+1}\times\mathbb{R})\;\simeq_{\ell i}\;\mathbb{R}^{m+2},
  \qquad
  \texttt{flattenCLE } m : \mathbb{R}^{m+2}\;\simeq_{L}\;(\mathbb{R}^{m+1}\times\mathbb{R}),
\]
built from an orthonormal-basis identification.  \texttt{flatten} is automatically
\texttt{MeasurePreserving} for \texttt{volume}, and by hand (via
\texttt{Isometry.euclideanHausdorffMeasure\_image}) for $\mu_H^{[m+1]}$; \texttt{flattenCLE} is the
continuous-linear equivalence used to conjugate the field.  Transferring both sides of the graph
theorem through \texttt{flatten} gives the flat theorem:

\begin{lstlisting}[caption={Flat-coordinate divergence theorem}]
theorem divergence_theorem_flat {m : N} {gamma} (hg : ContDiff R 1 gamma)
    (hg0 : ∀ x, 0 ≤ gamma x) {F} (hF : ContDiff R 1 F) (hsupp : HasCompactSupport F) :
    (∫ z in flatten m '' {p | (ofLp p).2 ∈ Ioo 0 (gamma (ofLp p).1)},
        divergenceE (fun w => (flattenCLE m).symm (F (flattenCLE m w))) z)
      = (∫ z in flatten m '' (graphFun gamma '' univ),
            <(flattenCLE m).symm (F (flattenCLE m z)),
                flatten m (graphNormal gamma ((flatten m).symm z).ofLp.1)>_R ∂muHE[m+1])
          - ∫ x, (F (x, 0)).2
\end{lstlisting}

The pointwise divergence bridge is \texttt{divergenceE\_flatten}:
$\operatorname{div}_E(\Phi^{-1}\!\circ F\circ\Phi)(\texttt{flatten }q)=\operatorname{div}(F)(\texttt{ofLp }q)$
with $\Phi=\texttt{flattenCLE}$, a one-line trace conjugation.  This intermediate form still carries
the finite window $\{0<\text{height}<\gamma\}$ and the hypothesis $\gamma\ge0$, both of which the
next step removes.

\subsection{6.5 The bottomless subgraph theorem (arbitrary $\gamma$)}

The public assembly needs the divergence theorem over the \emph{whole} strict subgraph
$\{\text{height}<\gamma(\text{base})\}$, and it needs it for \emph{arbitrary} $\gamma$: a boundary
chart centered at a boundary point $x_0$ has $\gamma(0)=0$, so $\gamma$ genuinely dips negative
nearby, and any $\gamma\ge0$ hypothesis is fatal.  The resolution is to work with the
\emph{iterated} integral and the oriented additivity of the one-dimensional integral,
\[
  \int_{(-\infty,\,b]} g \;=\;\int_{(-\infty,\,0]} g \;+\;\int_0^{b} g\qquad(\text{any sign of }b),
\]
so that the full subgraph splits as (half-space slab) $+$ (finite window) with the window handled by
the \emph{signed} graph theorem of \S6.3 --- neither piece needs $\gamma\ge0$.

\begin{lstlisting}[caption={Oriented Iic-additivity, Fubini below a graph, and the bottomless theorem}]
theorem integral_Iic_split {g : R -> R} (hg : Integrable g) (b : R) :
    ∫ t in Iic b, g t = (∫ t in Iic (0:R), g t) + ∫ t in (0:R)..b, g t

theorem setIntegral_below {m : N} {gamma} (hg : Measurable gamma) {h} (hint : ...) :
    ∫ p in {p | p.2 < gamma p.1}, h p ∂volume = ∫ x, ∫ t in Iio (gamma x), h (x, t)

theorem divergence_theorem_flat_bottomless {m : N} {gamma} (hg : ContDiff R 1 gamma)
    {F} (hF : ContDiff R 1 F) (hsupp : HasCompactSupport F) :
    (∫ z in flatten m '' {p | (ofLp p).2 < gamma (ofLp p).1},
        divergenceE (fun w => (flattenCLE m).symm (F (flattenCLE m w))) z)
      = ∫ z in flatten m '' (graphFun gamma '' univ),
          <..., flatten m (graphNormal gamma ((flatten m).symm z).ofLp.1)>_R ∂muHE[m+1]
\end{lstlisting}

The proof transfers the volume side to the iterated form
$\int_x\int_{(-\infty,\gamma x)}\operatorname{div}F$ (via \texttt{setIntegral\_below}), splits each
inner integral by \texttt{integral\_Iic\_split}, and recognizes the two resulting pieces as the slab
$\int_x\int_{(-\infty,0]}\operatorname{div}F=\int_x F_2(x,0)$
(\texttt{integral\_Iic\_divergence\_eq}) and the graph theorem
$\int_x\int_0^{\gamma x}\operatorname{div}F=\text{flux}-\int_x F_2(x,0)$; the two boundary terms
cancel and only the flux survives.  \texttt{integral\_Iic\_split} itself is a rearrangement of
Mathlib's \texttt{intervalIntegral.integral\_Iic\_sub\_Iic}.

\begin{remark}[A day lost to a parser, not to mathematics]
The naive statement \texttt{Iic 0, g + $\int$ 0..b, g} parses as \texttt{Iic 0, (g + $\int$ 0..b, g)}
--- the integral binder has very low precedence and silently swallows the \texttt{+}.  For a long
time this masqueraded as a bogus ``\texttt{$\int$ default} vs \texttt{$\int$ $\partial$volume} are
not defeq'' failure.  Two lessons for the file: always parenthesize
\texttt{($\int$ x in s, f x) + \dots}; and to align a Mathlib lemma stated with $\partial\mu$ to a
default-measure goal, bind it with an explicit type ascription (\texttt{have h : <default form> := lemma})
so the definitional check does the conversion, then close with \texttt{rw [<- h]; ring}.
\end{remark}

\subsection{6.6 Chart flux: the rigid-motion transfer}

A boundary chart presents $\Omega$ near $x_0$ as the subgraph in the rotated/translated frame
$x\mapsto\texttt{flatten}^{-1}(e(x-c))$.  Let $A(x)=e(x-c)$ be that affine isometry.  A field $V$
living in physical coordinates is \emph{actively transported} to $G(z)=e\big(V(e^{-1}z+c)\big)$: the
outer $e$ is essential, because a passive transport $V\circ A^{-1}$ would \emph{not} preserve the
divergence, whereas conjugation does,
\[
  \operatorname{div}_E G\big(A(x)\big)=\operatorname{tr}\!\big(e\,DV(x)\,e^{-1}\big)=\operatorname{div}_E V(x).
\]
Changing variables under $A$ on the volume integral, and under $A$ again on the $\mu_H$ surface
integral (both are isometries, hence measure-preserving via
\texttt{measurePreserving\_rigid\_muHE}), turns \S6.5 into the single-chart flux identity.  The
inner-product transport $\langle e\,a,w\rangle=\langle a,e^{-1}w\rangle$ produces the outward normal
$\nu(x)=e^{-1}\big(\texttt{flatten}(\texttt{graphNormal}\,\gamma\,(\operatorname{base}(x)))\big)$.

\begin{lstlisting}[caption={Divergence under an affine isometry, and chart flux}]
theorem divergenceE_transport_affine (e : R^n =li R^n) (c) (hV : Differentiable R V) (x) :
    divergenceE (fun z => e (V (e.symm z + c))) (e (x - c)) = divergenceE V x

theorem chart_flux (e : R^(m+2) =li R^(m+2)) (c) {gamma} (hg : ContDiff R 1 gamma)
    {V} (hV : ContDiff R 1 V) (hVsupp : HasCompactSupport V) :
    (∫ x in {x | height(x) < gamma (base x)}, divergenceE V x)
      = ∫ x in (fun x => e (x-c))⁻¹' (flatten m '' (graphFun gamma '' univ)),
          <V x, e.symm (flatten m (graphNormal gamma (base x)))>_R ∂muHE[m+1]
\end{lstlisting}

\subsection{6.7 The boundary is locally a graph}

To connect \texttt{chart\_flux} (whose sets are the full physical subgraph and graph) to the actual
domain $\Omega$, one needs $\operatorname{frontier}\Omega\cap B = \operatorname{graph}\cap B$ inside
each chart ball $B$.  This is purely topological.  In the plain product, the frontier of the open
subgraph is the graph, because the height coordinate is free:

\begin{lstlisting}[caption={Frontier of a subgraph is the graph; locality; and the domain version}]
theorem frontier_subgraph_prod {m : N} {gamma} (hg : Continuous gamma) :
    frontier {q : R^(m+1) x R | q.2 < gamma q.1} = {q | q.2 = gamma q.1}

theorem frontier_inter_open_eq (hU : IsOpen U) (h : s ∩ U = t ∩ U) :
    frontier s ∩ U = frontier t ∩ U

theorem chart_frontier_domain (e) (c) {gamma} (hg : Continuous gamma)
    (hchart : Om ∩ ball c r = {subgraph} ∩ ball c r) :
    frontier Om ∩ ball c r = {x | height x = gamma (base x)} ∩ ball c r
\end{lstlisting}

The $\subseteq$ direction of \texttt{frontier\_subgraph\_prod} is Mathlib's
\texttt{frontier\_lt\_subset\_eq}; the $\supseteq$ direction approximates each graph point
$(q_1,\gamma q_1)$ from below by $(q_1,\gamma q_1-\tfrac1n)$.  The level set is lifted to physical
coordinates through the chart homeomorphism $\Psi(x)=(\texttt{flatten}^{-1}(e(x-c))).\texttt{ofLp}$
with \texttt{Homeomorph.preimage\_frontier}, and \texttt{frontier\_inter\_open\_eq} (frontier is a
local notion on an open set) transfers the identity from the subgraph to $\Omega$ itself, using only
the chart equation $\Omega\cap B=\{\text{subgraph}\}\cap B$.

\subsection{6.8 The chart term}

Combining \S6.6 and \S6.7 with two support-localization steps gives the divergence theorem for a
field supported inside one chart ball --- but now phrased over $\Omega$ and $\partial\Omega$:

\begin{lstlisting}[caption={Single-chart divergence theorem in the domain}]
theorem chart_term (e) (c) {gamma} (hg : ContDiff R 1 gamma) {Om} (hOm : MeasurableSet Om) {r}
    (hchart : Om ∩ ball c r = {subgraph} ∩ ball c r) {nu}
    (hnu : ∀ x ∈ frontier Om ∩ ball c r, nu x = e.symm (flatten (graphNormal gamma (base x))))
    {V} (hV : ContDiff R 1 V) (hVsupp : HasCompactSupport V) (hVball : tsupport V ⊆ ball c r) :
    ∫ x in Om, divergenceE V x = ∫ x in frontier Om, <V x, nu x>_R ∂muHE[m+1]
\end{lstlisting}

The support helper is \texttt{setIntegral\_eq\_of\_support\_subset}: $\int_s f=\int_t f$ whenever
$\operatorname{supp}f\subseteq B$ and $s\cap B=t\cap B$ (an indicator-function argument).  It localizes
the volume integral from $\Omega$ to the subgraph and the flux integral from the graph to
$\partial\Omega$; the chart normal is identified with $\nu$ on $\partial\Omega\cap B$ by hypothesis
\texttt{hnu}, and \texttt{physical\_graph\_eq} identifies \texttt{chart\_flux}'s preimage-graph with
the level set $\{\text{height}=\gamma\}$.

\subsection{6.9 The domain, the normal, and the partition of unity}

\begin{lstlisting}[caption={Bounded C1 domain and outward normal}]
structure IsBoundedC1Domain {m : N} (Om : Set (R^(m+2))) : Prop where
  isOpen    : IsOpen Om
  isBounded : Bornology.IsBounded Om
  locallyGraph : ∀ x0 ∈ frontier Om, ∃ r > 0, ∃ (e : R^(m+2) =li R^(m+2)) gamma, ContDiff R 1 gamma ∧
    Om ∩ ball x0 r = {x | height x < gamma (base x)} ∩ ball x0 r

structure IsOutwardNormal {m : N} (Om : Set (R^(m+2))) (nu : R^(m+2) -> R^(m+2)) : Prop where
  continuous : Continuous nu
  eq_chart : ∀ c r (e : R^(m+2) =li R^(m+2)) gamma, ContDiff R 1 gamma ->
    Om ∩ ball c r = {subgraph} ∩ ball c r ->
    ∀ x ∈ frontier Om ∩ ball c r, nu x = e.symm (flatten m (graphNormal gamma (base x)))
\end{lstlisting}

$\partial\Omega$ is compact (a closed subset of $\overline\Omega$, which is compact because $\Omega$
is bounded), so it is covered by \emph{finitely many} chart balls
(\texttt{exists\_finite\_chart\_cover}, from \texttt{IsCompact.elim\_finite\_subcover}).  On that
cover Mathlib's \texttt{SmoothPartitionOfUnity.exists\_isSubordinate} produces a smooth partition of
unity $\{f_i\}_{i:\mathtt{Option}\,\iota}$ on $\overline\Omega$ subordinate to
$\{\Omega\}\cup\{\text{balls}\}$ (\texttt{exists\_smoothPartitionOfUnity}): the index \texttt{none}
is the interior patch, \texttt{some }$j$ is the $j$-th boundary chart.  \texttt{IsOutwardNormal}
requires $\nu$ to agree with the chart normal in \emph{every} valid subgraph chart, so it applies to
each cover chart directly.

\subsection{6.10 The partition-of-unity assembly}

Set $V_i=f_i\cdot F$.  Because $f_i$ is a bundled smooth map, its $C^1$-ness comes from
\texttt{contMDiff\_iff\_contDiff}, and $\operatorname{tsupport}f_i\subseteq U_i$ is a \emph{bounded}
closed set, hence compact --- so each $V_i$ is $C^1$ with compact support \emph{even though} $F$ is
only $C^1$.  On the open set $\Omega$ the partition sums to one, so $F=\sum_i V_i$ there and
(comparing Fréchet derivatives on a neighborhood, \texttt{Filter.EventuallyEq.fderiv\_eq})
$\operatorname{div}_E F=\sum_i\operatorname{div}_E V_i$ on $\Omega$.  Hence
\[
  \int_\Omega\operatorname{div}_E F
    \;=\;\sum_i\int_\Omega\operatorname{div}_E V_i .
\]
The $i=\mathtt{none}$ term vanishes: $V_{\mathtt{none}}$ is supported in $\Omega$, so
$\int_\Omega\operatorname{div}_E V_{\mathtt{none}}=\int_{\mathbb{R}^{m+2}}\operatorname{div}_E
V_{\mathtt{none}}=0$ (\texttt{integral\_divergenceE\_eq\_zero}), and its boundary flux is zero because
$V_{\mathtt{none}}=0$ on $\partial\Omega$ (which is disjoint from the open $\Omega$).  Each
$i=\mathtt{some}\,j$ term is exactly \texttt{chart\_term}.  Reassembling,
\[
  \sum_i\int_{\partial\Omega}\langle V_i,\nu\rangle
    \;=\;\int_{\partial\Omega}\Big\langle\textstyle\sum_i V_i,\nu\Big\rangle
    \;=\;\int_{\partial\Omega}\langle F,\nu\rangle ,
\]
using $\sum_i V_i=F$ on $\partial\Omega\subseteq\overline\Omega$ and \texttt{sum\_inner}.  The
exchange of $\sum$ and $\int$ needs each $\langle V_i,\nu\rangle$ integrable against
$\mu_H\!\restriction\!\partial\Omega$, which follows from \S6.11.

\subsection{6.11 Finiteness of the surface measure}

That last integrability --- and the removal of any hypothesis about it --- rests on
$\mu_H^{[m+1]}(\partial\Omega)<\infty$.  This is a genuine theorem, proved from the same finite chart
cover and the area formula.

\begin{lstlisting}[caption={The boundary has finite surface measure}]
theorem surfaceMeasure_graphSet_lt_top (e) (c) {gamma} (hg : ContDiff R 1 gamma) (r : R) :
    (muHE[m+1] : Measure (R^(m+2))) ({x | height x = gamma (base x)} ∩ ball c r) < ⊤

theorem surfaceMeasure_frontier_lt_top {m : N} {Om} (hOm : IsBoundedC1Domain Om) :
    (muHE[m+1] : Measure (R^(m+2))) (frontier Om) < ⊤
\end{lstlisting}

For one chart, the graph piece inside the ball projects into the base ball of radius $r$
($\|\operatorname{base}(x)\|\le\|x-c\|<r$ by \texttt{WithLp.norm\_fst\_le} and the chart isometries),
so it is contained in $A^{-1}\big(\texttt{flatten}(\operatorname{graph}\gamma''\,\overline{B}(0,r))\big)$;
transporting $\mu_H$ back through the rigid motion and \texttt{flatten} and invoking the area formula
gives $\mu_H\le\int_{\overline B(0,r)}\sqrt{1+\|\nabla\gamma\|^2}$, which is at most (sup of the
continuous integrand on the compact ball)$\times\operatorname{vol}(\overline B(0,r))<\infty$.  For the
whole boundary, $\partial\Omega=\bigcup_j(\partial\Omega\cap B_j)$ over the finite cover, and
\texttt{measure\_iUnion\_le} plus \texttt{chart\_frontier\_domain} reduce to the per-chart bound.
With $\mu_H^{[m+1]}(\partial\Omega)<\infty$ the restricted measure is finite, so every continuous
compactly-supported integrand --- in particular each $\langle V_i,\nu\rangle$ --- is integrable, and
the hypothesis-free \texttt{divergence\_theorem} of \S6.0 follows.

\subsection{6.13 Why the formal proof is long, and how Mathlib could shorten it}

\paragraph{Comparison with the textbook.}
Evans does \emph{not} prove this theorem: the Gauss--Green formula appears in Appendix~C.2
(Theorem~1) as a \emph{cited} known result under the single hypothesis that $\partial U$ is $C^1$.
Even a genuine proof is only a few pages, because it takes an entire lower edifice \emph{as given}.
That edifice is exactly what the formalization must build --- which is where all the length lives:

\begin{center}
\begin{tabular}{@{}L{4.6cm}L{3.2cm}L{6.4cm}@{}}
  \toprule
  \textbf{Ingredient the proof needs} & \textbf{In Evans} & \textbf{In this formalization} \\
  \midrule
  the Gauss--Green formula itself & stated, not proved (App.~C.2) & \textbf{proved} --- \texttt{divergence\_theorem} (\S6.0) \\
  surface measure $dS$ on $\partial\Omega$ & assumed known & built: $\mu_H$ $+$ area formula (\S6.2) \\
  area / change-of-variables formula & assumed known & \textbf{built from scratch} --- the largest single block (\S6.2) \\
  multivariate integration by parts & assumed known & built: horizontal-IBP toolchain (\texttt{Calculus.\allowbreak lean}) \\
  partition of unity & assumed known & Mathlib \texttt{SmoothPartitionOfUnity} $+$ bridge (\S6.10) \\
  existence of the outward normal & assumed known & assumed (\texttt{IsOutwardNormal} hypothesis) \\
  \bottomrule
\end{tabular}
\end{center}

\noindent So the raw size --- \texttt{AreaFormula.\allowbreak lean} $\approx$2870 lines across $118$
declarations, plus $\approx$270 lines of horizontal IBP in \texttt{Calculus.\allowbreak lean} ---
reflects \emph{not} greater mathematical depth but the missing foundation. The divergence theorem
\emph{proper} (the chart term and partition-of-unity assembly, \S6.8--6.10) is a thin capstone; the
bulk is the surface-measure layer beneath it (\S6.2, \S6.4) that a textbook disposes of in one
citation. In short: the formalization is long because it must \emph{construct the foundations the
book assumes}.

\paragraph{Which basic Mathlib tools would collapse it.}
With any of the following in Mathlib the proof would shrink to a fraction of its size; they are
listed by impact, with the parts of this chapter they would eliminate.

\begin{enumerate}
  \item \textbf{An area / coarea formula for $C^1$ maps} (or even just $\mu_H$ of a $C^1$ graph
    $=\int\sqrt{1+\|\nabla\gamma\|^2}$). This removes \S6.2 outright --- the largest single block,
    currently built by covering into near-linear cells via the inverse function theorem, applying
    the linear image formula on each cell, and passing to the limit --- and makes the
    surface-measure finiteness of \S6.11 immediate.
  \item \textbf{A first-class \texttt{EuclideanSpace} product/concatenation isometry}
    $\mathbb{R}^a\times\mathbb{R}^b\simeq_{\ell i}\mathbb{R}^{a+b}$ registered as volume- and
    $\mu_H$-preserving. The graph computation is forced onto the plain product
    $\mathbb{R}^{m+1}\times\mathbb{R}$, which carries the \emph{sup norm} and is \emph{not} an
    inner-product space; that is the sole reason for the internal second divergence \texttt{divergence}
    and for the hand-built isometry \texttt{flatten}/\texttt{flattenCLE}. This tool deletes that entire
    duality: \S6.4 (\texttt{divergence\_theorem\_flat}, \texttt{divergenceE\_flatten}) and the
    \texttt{flatten}-transport steps in \S6.6 and \S6.11 disappear.
  \item \textbf{Multivariate integration by parts} --- a usable
    ``$\int_{\mathbb{R}^n}\partial_{x_i}(fg)=0$ for compactly supported $C^1$''. This erases the
    whole horizontal-IBP toolchain in \texttt{Common/\allowbreak Calculus.\allowbreak lean}
    (compact-support FTC, the $\mathrm{Fin.insertNth}$ coordinate-slice lemmas, the Fubini lift, and
    the three \texttt{integral\_horizontal\_ibp} forms) that supplies the $\partial_{x_i}$ half of the
    graph theorem (\S6.3) and \texttt{integral\_divergenceE\_eq\_zero} (\S6.1).
  \item \textbf{The $C^1$ implicit function theorem for level sets / a hypersurface normal bundle.}
    This is the one tool that would improve the \emph{statement}, not merely the proof: it would let
    the theorem \emph{construct} the outward normal from the charts and prove it exists, so
    \texttt{IsOutwardNormal} could be dropped as a hypothesis and \texttt{divergence\_theorem} would
    take only $(\Omega,F)$ (cf.\ the Scope remark in \S6.0).
  \item \textbf{Ergonomic support:} $\mu_H$ isometry-invariance for the \emph{normalized} measure
    (removing the hand proofs \texttt{measurePreserving\_rigid\_μHE} and
    \texttt{flatten}.\allowbreak\texttt{euclideanHausdorffMeasure\_image}, \S6.4/6.6/6.11), and a
    $C^1$ partition of unity subordinate to an open cover of a compact set returning \texttt{ContDiff}
    functions directly (cleaning up the \texttt{SmoothPartitionOfUnity}-over-\texttt{Option} bridge of
    \S6.10).
\end{enumerate}

Items 1--3 are the three from-scratch pillars; supplying them would roughly halve the development.
Item~4 would strengthen the theorem itself. A full \textbf{Stokes theorem for $C^1$
manifolds-with-boundary} would of course subsume the entire \S6.3--6.10 pipeline, but that is less a
\emph{tool} than the endpoint --- it is Gauss--Green already.
